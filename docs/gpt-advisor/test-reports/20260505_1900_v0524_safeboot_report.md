# P0-STARTUP-SAFEBOOT-001 / v0.5.24 Safe Boot 报告

时间：2026-05-05 19:00

## 结论

本轮停止新增定位、历史持久化、图片预览功能，只处理 P0 启动回归。v0.5.24 建立 Safe Boot 启动顺序，目标是任何 Bridge、G2、历史、定位、交易、OpenCLAW、ASR 失败都不能阻塞手机端菜单点击。

## 本轮改动

- `main()` 不再在按钮绑定前 `await initBridge()`。
- 手机端静态 UI、默认书签、全局反馈、关键视觉按钮、书签事件代理全部提前执行。
- 新增早期事件代理：`.bookmark-tabs` 点击通过 `[data-bookmark]` 切换手机页面。
- `initBridge()`、G2 首页、设备电量、R1/G2 事件注册移入 `bootBridgeAndGlassInBackground()`。
- `renderHistory()` 移入 `hydrateHistoryInBackground()`。
- `prepareVoiceInput()` 移入 `initOptionalDiagnosticsInBackground()`。
- G2 启动失败只记录 warning 和状态，不影响手机菜单。

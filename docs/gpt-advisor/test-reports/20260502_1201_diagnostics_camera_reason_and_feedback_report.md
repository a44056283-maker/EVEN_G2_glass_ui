# 诊断原因与按钮反馈修复报告

时间：2026-05-02

## 本轮修复

- 诊断中心不再只显示“相机：请求失败”，改为相机失败原因分类。
- 相机失败会尽量归类为：权限拒绝、无相机、被占用、流未就绪、容器限制、未知原因。
- 网页端按钮增加可感知反馈：按压缩放、高亮发光、状态 toast、支持时轻震动。
- R1 事件增加状态反馈：单击/双击/上滑/下滑会在网页状态区显示。
- 相机底层错误不再统一伪装成“允许相机权限”，而是保留 WebView/getUserMedia 的真实失败信息。

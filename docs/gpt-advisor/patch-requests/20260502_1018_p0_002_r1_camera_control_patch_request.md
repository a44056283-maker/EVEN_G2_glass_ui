# P0-002 R1 相机控制与眼镜菜单确认整改任务

创建时间：2026-05-02 10:18

## 前置结果

用户真机反馈：

```text
手机网页可以动手拍照、上传并识别。
```

说明 `P0-001 网页视觉识别闭环` 已通过真机基础验证。

## 本轮范围

本轮只修复：

1. R1 能在眼镜端进行菜单选择和确认进入下一级菜单。
2. R1 能在视觉识别页控制相机：
   - 进入视觉页后预热手机相机。
   - R1 单击拍照。
   - R1 双击或确认发送图片识别。
   - R1 上滑重拍。
   - R1 下滑取消/返回。

## 本轮禁止修改

- G2 麦克风
- ASR
- OpenCLAW
- 交易机器人接口
- 复杂 Glass UI 大改
- 手机网页视觉美化

## 验收标准

- [ ] R1 在首页上下切换一级菜单。
- [ ] R1 单击视觉识别后进入视觉下一级页，而不是停在首页。
- [ ] 进入视觉页后手机相机预热。
- [ ] R1 单击可以拍照。
- [ ] 拍照后 G2 显示“已拍照/发送/重拍/取消”提示。
- [ ] R1 双击或确认可以发送图片识别。
- [ ] 识别结果显示到网页和 G2。
- [ ] 不影响 P0-001 手机网页手动拍照识别。

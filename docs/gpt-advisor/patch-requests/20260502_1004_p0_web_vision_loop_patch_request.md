# P0-001 网页视觉识别闭环整改任务

创建时间：2026-05-02 10:04

## 本轮范围

只修复手机网页上的视觉识别基础闭环：

```text
首页点击“视觉识别”
-> 只进入视觉识别子版块
-> 不直接拍照
-> 用户点击“拍照识别 / 直接拍照”
-> 获取图片
-> POST https://g2-vision.tianlu2026.org/vision
-> 手机网页显示识别结果
-> G2 眼镜端显示简短识别结果
-> TTS 失败也不影响文字结果显示
```

- ASR

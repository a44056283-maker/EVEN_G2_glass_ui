# 隐私与上传策略

## V0

- 只在用户点击后拍照上传。
- 不连续上传视频流。
- API key 只保存在后端 `.env`。
- 前端不保存 MiniMax API key。
- 失败时使用本机浏览器朗读，不把额外语音上传。

## 后续视频抽帧版

视频版本也不直接上传 30fps 视频流，而是：

- 每 3-5 秒抽帧
- 做画面变化检测
- 变化明显才上传
- 最多每 10 秒上传一次
- 只保留最近 5 次视觉摘要作为场景记忆

## 语音版

语音问答必须默认 push-to-talk，只在用户主动触发时上传音频。

## 只读记忆检索

- 只检索用户授权同步到本机的记忆缓存。
- 不同步 `.env`、key、token、secret、credential、webhook、pem 等敏感文件。
- 记忆检索只用于回答用户问题，不直接修改原始记忆库。
- 涉及交易系统、仓位、策略、V6.5 等内容时，只做只读分析、复盘、风险提示和建议清单。
- G2 Vision AI 不执行自动开仓、平仓、改仓、下单或交易接口调用。

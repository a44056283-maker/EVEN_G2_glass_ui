## 降噪判断来源

### 1. 趋势突破

用于识别假突破和无量突破:

- 量能未达到 5.0x 且 M3 最大放量未达到 7.0x -> 噪音加分。
- 放量达标 -> 噪音降分。

### 2. 资金流异动

用于避免方向反向:

- 做多但 M1 净流出 -> 噪音加分。
- 做空但 M1 净流入 -> 噪音加分。
- 做多且 M1 净流入 / 做空且 M1 净流出 -> 噪音降分。

### 3. 支撑压力确认

用于避免压力做多、支撑做空:

- 做多靠近压力且远离支撑 -> 噪音加分。
- 做空靠近支撑且远离压力 -> 噪音加分。
- 做多靠近支撑 / 做空靠近压力 -> 噪音降分。

### 4. OI/爆仓/盘口

用于识别追涨杀跌和盘口反向:

- 做多但盘口卖盘倾斜 -> 噪音加分。
- 做空但盘口买盘倾斜 -> 噪音加分。
- 多头爆仓后追多、空头爆仓后追空 -> 噪音加分。

### 5. 做多做空差异

多空使用同等权利,但方向判断独立:

- LONG 检查支撑、净流入、买盘倾斜、巨牛量。
- SHORT 检查压力、净流出、卖盘倾斜、巨熊量。
- 不允许出现只利好多、不利空的单边偏见。

- 样本 >= 300。
- 连续运行 >= 7 天。
- 高噪音标签能覆盖亏损/反向/假突破信号。
- 优质标签不能明显错过 V6.5 有效入场。
- 不增加过早平仓。
- 不改变 V6.5 的开仓权重,只作为质量分层。

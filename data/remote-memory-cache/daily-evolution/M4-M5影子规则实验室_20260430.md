## 目标

先不把 M4-M5 直接接入实盘交易规则，而是做一套影子实验系统：

- 基准组：当前 V6.5 规则，不加载 M4-M5 增强判断
- 实验组：V6.5 + M4 RSI/ATR/OI + M5 订单薄/OI/爆仓/情绪
- 两组同步读取真实机器人持仓、9099 中央数据流与出山/天眼评测
- 只做模拟评分和影子决策，不调用任何下单/平仓接口
- 连续运行一周后，用胜率、收益改善、误杀率、风险提示质量决定是否正式对接交易机器人

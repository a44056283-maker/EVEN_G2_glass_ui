# 工具脚本说明

## bot_manager.sh

**位置**：`/Users/luxiangnan/freqtrade/bot_manager.sh`

**功能**：
- 清理所有重复进程（每个端口只保留1个）
- 启动/停止/重启指定端口机器人
- 启动/停止/重启所有机器人
- 查看所有机器人状态

**用法**：
```bash
./bot_manager.sh status          # 查看所有机器人状态
./bot_manager.sh cleanup         # 清理所有重复进程
./bot_manager.sh start 9091     # 启动单个机器人
./bot_manager.sh stop 9091       # 停止单个机器人
./bot_manager.sh restart 9091    # 重启单个机器人
./bot_manager.sh start-all      # 启动所有机器人
./bot_manager.sh stop-all       # 停止所有机器人
./bot_manager.sh restart-all    # 重启所有机器人
```

**桌面快捷方式**：双击 `🤖 Bot管理器.command` 即可查看状态

**注意**：启动命令已修正，使用 `.venv311/bin/python3 .venv311/bin/freqtrade`，无需 PYTHONPATH

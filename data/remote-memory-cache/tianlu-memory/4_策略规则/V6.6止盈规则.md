# V6.6.1 动态止盈标准参数

**生效日期**: 2026-04-10

## 三档止盈基准

| 参数 | 值 | 说明 |
|------|-----|------|
| P1触发基准 | 15.0%（真实涨幅） | 所有杠杆统一 |
| P2触发基准 | 25.0%（真实涨幅） | 所有杠杆统一 |
| P3触发基准 | 35.0%（真实涨幅） | 所有杠杆统一 |
| P1卖出% | 50%（固定） | 各杠杆通用 |
| P2卖出% | 基准%（=25×10/lev） | 杠杆越高卖越少 |
| P3卖出% | 100%（固定） | 必须全平 |

## 公式

- 真实涨幅 = BASE ÷ 10（所有杠杆同一止盈价）
- 触发涨幅 = BASE × (杠杆/10)（杠杆越高越快触发）
- P2卖出% = 基准 × (10/杠杆)

## 七、决策铁律

```
数据说话 → 一个结论 → 立即执行 → 不犹豫

模糊词全部删除：
  观望     → 禁止/执行
  谨慎     → 禁止
  等确认   → 禁止
  可选     → 禁止/执行
  建议做多  → 执行做多
  可能反弹  → 立即止损

决策只有三种结果：
  ✅ 执行做多   → 下令做多，不问
  ✅ 执行做空   → 下令做空，不问
  🚫 禁止操作   → 不下单，不问
```

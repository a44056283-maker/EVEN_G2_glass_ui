## 👑 太子（天禄）· 审批审查流程

### 职责
- 接收各部提交的审议申请
- 评估事项紧急程度
- 重要事项提报父亲（爸）审批

### 审批流程
```
各部审议 → 太子审查 → 重要事项提报 → 父亲审批 → 执行
                       ↓
                 一般事项直接决定
```

### 涉及脚本
| 脚本 | 触发条件 | 处理方式 |
|------|---------|---------|
| `monitor_sentiment.py` | 黑天鹅警报 | 太子审查 → 提报父亲 |
| `audit_guard.py` | 刑部复核 | 太子审查 → 提报父亲 |
| `monitor_8x_leverage.py` | 8倍杠杆警告 | 太子审查 → 提报父亲 |
| `real_time_guard.py` | 实时异常 | 太子审查 → 直接处理或提报 |

### 紧急度判断
- 🔴 **紧急**（urgency≥8 或 severity=critical）→ 立即提报父亲
- 🟡 **一般**（urgency 5-7）→ 太子决定
- 🟢 **可忽略**（urgency<5）→ 静默处理

## 三、增强方案

### 3.1 L0 增强：实时强平价计算 + VaR 监控

**新增功能：**
- 实时计算多仓/空仓强平价：`多仓 = 开仓价 × (1 - 1/杠杆)` / `空仓 = 开仓价 × (1 + 1/杠杆)`
- 实时计算距强平距离百分比
- 95% 置信度 VaR 监控（基于持仓历史）

**新增阈值：**
```python
LIQUIDATION_ALERT_P0 = 5.0    # 距强平 < 5%  → P0
LIQUIDATION_ALERT_P1 = 10.0   # 距强平 < 10% → P1
LIQUIDATION_ALERT_P2 = 20.0   # 距强平 < 20% → P2
VAR_95_THRESHOLD_PCT = 3.0     # VaR 95% > 3% → P1 告警
```

**预期效果：** 全仓位的实时强平距离可见，不仅仅是 SOL/DOGE。

---

### 3.2 L3 增强：乾天监情感数据接入 Trigger

**新增功能：**
- 乾天监 `sentiment_pool.json` → 兵部监控中枢 → L3 Trigger 第9层（综合风险评级）
- Fear & Greed 中文解读 + 策略建议
- 社区情绪（Reddit）综合评分接入

**Fear & Greed 策略建议：**
| 值域 | 中文解读 | 策略建议 |
|------|---------|---------|
| 0-25 | 极度恐慌 | 可分批布局，等待反转确认 |
| 25-45 | 恐慌 | 谨慎做多，设置严格止损 |
| 45-55 | 中立 | 观望，等待趋势明确 |
| 55-75 | 贪婪 | 持有多头，注意利润保护 |
| 75-100 | 极度贪婪 | 分批止盈，不追高 |

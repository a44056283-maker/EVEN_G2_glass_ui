# 三省六部 工作日程系统 V2.0

## 📅 每日朝会流程

```
🌅 卯时（08:00）- 各部报到
  各部门大臣向尚书省报到，汇报今日工作计划

🌅 巳时（10:00）- 任务派发
  尚书省发布今日任务清单

🌅 午时（12:00）- 中盘汇报
  各部汇报上午工作进展

🌅 酉时（18:00）- 终盘汇报
  各部汇报今日完成情况

🌅 戌时（20:00）- 尚书省汇总
  尚书省汇总全天工作，向太子汇报
```

## 📋 各部工作日程

### 🏛️ 中书省（策略起草）- 主管策略研发

| 时间 | 任务 | 说明 |
|------|------|------|
| 08:30 | 📊 策略表现分析 | 分析昨日策略盈亏，找出优化点 |
| 10:00 | 📝 策略参数调整 | 根据分析结果调整参数 |
| 14:00 | 🔬 回测验证 | 测试新策略效果 |
| 16:00 | 📋 提案起草 | 撰写明日策略改进提案 |
| 18:00 | ✅ 向上汇报 | 向尚书省汇报今日工作 |

**执行脚本**: `param_optimizer.py`, `apply_model_changes.py`

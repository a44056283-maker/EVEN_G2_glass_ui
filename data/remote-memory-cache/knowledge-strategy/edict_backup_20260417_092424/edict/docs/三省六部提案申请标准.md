# 三省六部 提案申请标准模板

## 📋 提案格式 V1.0

所有部门的提案申请必须按以下标准格式提交：

---

### 🏛️ 中书省（策略提案）

```
【提案类型】策略参数调整
【提案ID】ZZ-YYYYMMDD-XXX

📝 标题: [策略名称/参数调整]
📤 发起部门: 中书省
📥 目标部门: 门下省（审议）

📄 详细内容:
  • 当前参数: [参数名] = [当前值]
  • 建议调整: [参数名] = [新值]
  • 调整理由: [说明原因]
  • 预期效果: [预期收益/改善]
  • 风险评估: [低/中/高]
  • 回滚方案: [如有]

🔗 相关脚本: 
  - param_optimizer.py
  - taizi_triage.py

⏰ 创建时间: YYYY-MM-DD HH:MM
👤 发起人: 中书令
```

## 📝 统一字段说明

| 字段 | 说明 | 必填 |
|------|------|------|
| 提案ID | 格式: 部门缩写-年月日-序号 | ✅ |
| 标题 | 提案简短描述 | ✅ |
| 发起部门 | 发起部门名称 | ✅ |
| 目标部门 | 接收部门名称 | ✅ |
| 详细内容 | 具体内容说明 | ✅ |
| 相关脚本 | 涉及的脚本 | 可选 |
| 创建时间 | 自动生成 | ✅ |

## 🔧 自动化发送

使用 `send_task_notification.py` 发送标准化卡片消息，支持：
- ✅ 批准按钮
- ❌ 拒绝按钮  
- 📝 修改意见按钮

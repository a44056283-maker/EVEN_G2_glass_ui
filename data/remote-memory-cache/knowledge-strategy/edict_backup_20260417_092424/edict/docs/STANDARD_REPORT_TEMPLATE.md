## 五、汇报渠道与时间

| 汇报类型 | 发送时间 | 发送渠道 | 接收方 |
|----------|----------|----------|--------|
| 日报 | 每日 20:00 | 飞书/微信 | 尚书省备案 |
| 时报 | 事件发生时 | 飞书/微信 | 尚书省 + 相关部门 + 爸 |
| 周报 | 每周一 09:00 | 飞书文档 | 各省 + 爸 |
| 月报 | 每月1日 09:00 | 飞书文档 | 各省 + 爸 |

## 六、汇报质量标准

1. **及时性**：按时提交，不拖延
2. **准确性**：数据真实，不编造
3. **完整性**：要素齐全，不遗漏
4. **简洁性**：重点突出，不废话
5. **可操作性**：有结论有建议，不只陈述

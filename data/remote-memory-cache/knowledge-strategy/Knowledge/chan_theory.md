## 一、分型（最基础构件）

分型是缠论最小结构单元，由3根连续K线构成。

### 顶分型
```
第1根K线高点 > 第2根K线高点
第2根K线高点 > 第3根K线高点（但第3根可以更低）
```
特征：上升趋势中，多方力量衰竭

### 底分型
```
第1根K线低点 < 第2根K线低点
第2根K线低点 < 第3根K线低点（但第3根可以更高）
```
特征：下降趋势中，空方力量衰竭

### 识别规则
- 包含关系处理：两根K线有包含关系时，取 extremas 的 extremes
- 有效分型：需经包含处理后确认

---

## 二、笔（连接分型的线段）

笔 = 顶分型 + 至少1根独立K线 + 底分型（或反向）

### 笔的规则
1. 顶分型之后必然是底分型（笔终结）
2. 笔的幅度要求：顶底分型之间应有明显幅度（无明确数值要求，但需目视可辨）
3. 笔破坏：原笔方向被反向笔打破

### 笔的方向
- 上升笔：底分型起点 → 顶分型终点
- 下降笔：顶分型起点 → 底分型终点

---

## 三、线段（笔的叠加）

线段 = 连续3笔以上，同方向
- 线段被另一线段破坏，才算真正破坏
- 上升线段：连续3笔上升
- 下降线段：连续3笔下降

---

## 四、中枢（核心结构）

中枢 = 至少3段连续笔的重叠区间

```
         ┌─── 笔1 ───┐
    笔3 ←│  重叠区间  │→ 笔4
         └─── 笔2 ───┘
            ↑ 中枢区间
```

### 中枢的分类
- **上升中枢**：高低点逐步抬升
- **下降中枢**：高低点逐步下降
- **中枢扩展**：延伸出9段以上

### 中枢的实战意义
- 交易区间：中枢内高抛低吸
- 突破确认：离开中枢后的回撤不进入=有效突破
- 买卖点：一买/二买/三买；一卖/二卖/三卖

---

## 五、实战应用步骤

### Step 1: 找分型
```
观察K线群 → 标记顶分型和底分型 → 去除无效分型
```

### Step 2: 连笔
```
顶分型 → 独立K线 → 底分型 = 上升笔
底分型 → 独立K线 → 顶分型 = 下降笔
```

### Step 3: 找线段
```
3笔同方向 = 线段
线段被反向线段破坏 = 线段终结
```

### Step 4: 画中枢
```
找出连续3段笔的重叠区间
→ 标注为中枢区间
→ 观察价格与中枢的关系
```

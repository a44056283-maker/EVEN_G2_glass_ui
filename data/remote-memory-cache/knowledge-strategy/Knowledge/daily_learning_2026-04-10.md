## 一、模板设计原则

**为什么要可填写？**
- 昨天的学习建立了量化框架，今天要把它变成可直接使用的工具
- 光有理论不够，必须能在每笔交易后5分钟内完成自检
- 模板要足够简洁，复杂了就没人会用

**模板结构：**
1. 交易前检查（开仓前填写）
2. 交易后复盘（平仓后填写）
3. 周度偏差评分（每周日汇总）

## 二、交易前检查表（开仓前必填）

```
┌─────────────────────────────────────────────────────────────┐
│                    交易前逆向检查表                           │
├─────────────────────────────────────────────────────────────┤
│ 日期：_________  品种：_________  方向：多/空               │
│ 预期持仓时间：_________小时    计划仓位占比：_______%          │
├─────────────────────────────────────────────────────────────┤
│ 【逆向检查】完成以下3项才能开仓（全部勾选）                    │
│                                                             │
│ □ 如果现在不持有，我会买入吗？（不要因为持仓而做多）           │
│                                                             │
│ □ 该品种近期3个利空因素是什么？                               │
│   1. ________________________                               │
│   2. ________________________                               │
│   3. ________________________                               │
│                                                             │
│ □ 止损位是基于技术面还是成本价？                               │
│   技术依据：________________________（填技术位才过）          │
│   ❌仅填"成本价-2%"视为不通过                                 │
├─────────────────────────────────────────────────────────────┤
│ 逆向检查得分：____/30（≥20分才可开仓）                        │
└─────────────────────────────────────────────────────────────┘
```

**评分标准：**
- 3项全部勾选且技术依据具体：30分 ✓
- 3项勾选但技术依据模糊：20分 ⚠（附条件通过）
- 少于3项勾选：0分 ✗（禁止开仓）

├─────────────────────────────────────────────────────────────┤

│ 是否需要加入禁止清单：□是 □否                               │

## 五、模板使用指南

### 频率要求
- **交易前检查表**：每笔开仓前必填，电子化存档
- **交易后复盘表**：每笔平仓后24小时内填写
- **周度汇总表**：每周日20:00前完成

### 存档格式
```
~/trading_journal/
├── 2026-04/
│   ├── pre_trade_2026-04-10_BTC_LONG.md
│   ├── post_trade_2026-04-10_BTC_LONG.md
│   └── weekly_summary_2026-04-06_to_2026-04-10.md
```

### 关键阈值
| 总分 | 状态 | 行动 |
|------|------|------|
| 0-20 | 正常 | 无需干预 |
| 21-50 | 注意 | 当日停止开新仓 |
| 51-100 | 危险 | 强制冷静期3天，禁止开仓 |
| >100 | 极端 | 联系外部审计（家人/导师） |

## 七、核心要点（3个）

1. **逆向检查表是防止过度自信的第一道防线**：三个问题（现在会买吗？利空是什么？止损基于技术还是成本？）必须在开仓前完成，任何一个敷衍了事都是危险信号。技术依据栏禁止填"成本价"，只接受具体技术位。

2. **交易后复盘表将偏差量化落到操作层**：不是记录"今天心态不好"，而是明确"哪道检查题触发了偏差"。15分/12分/13分的分值设计对应实际危害程度，让复盘有优先级。

3. **周度追踪表比单笔复盘更重要**：单笔复盘容易陷入"这一笔该不该"的细节，而周度视角能看到"我是不是这周一直在犯同样的错"。趋势比单点数据更有诊断价值。

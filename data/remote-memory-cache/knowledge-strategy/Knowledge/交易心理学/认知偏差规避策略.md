## 一、五大认知偏差及其规避策略

### 1. 损失厌恶 + 处置效应

**定义**：损失的痛苦是同等收益快感的2倍，导致拒绝认赔、死守亏损仓

**在交易中的表现**：
- 浮亏时不设止损，认为"只要不卖就不算亏"
- 盈利时急于止盈，怕"到手的利润飞走"
- 止损触发后撤销，想"再等等看"

**规避策略**：

| 策略 | 操作 | 效果 |
|------|------|------|
| 硬止损 | 开仓即设止损单，不进终端手动操作 | 排除情绪干扰 |
| 盈亏比规则 | 止盈:止损 ≥ 2:1，坚持不变 | 盈利覆盖亏损 |
| 分批平仓 | 盈利先出一部分 | 降低持仓压力 |
| 非交易时间决策 | 盘后做决策，避免盘中情绪 | 冷静期 |

**实战口诀**：止损不是认输，是保命。

---

### 2. 过度自信 + 过度交易

**定义**：高估自己的预测能力，交易频率超过理性最优

**在交易中的表现**：
- 认为能"预测"短期价格，频繁进出
- 看到消息立刻追入，未做验证
- 同时持有5个以上山寨币，觉得"分散了风险"

**规避策略**：

| 策略 | 操作 | 效果 |
|------|------|------|
| 交易日志审查 | 每周检查交易频率 vs 胜率相关性 | 发现过度交易 |
| 策略手册 | 每个策略有书面入场/出场/风控规则 | 不随意改动 |
| 最大持仓数 | 山寨币≤3个，总仓位≤账户20% | 控制风险暴露 |
| 错误日志不过夜 | 技术问题当日解决，不累积 | 避免问题恶化 |
| 冷静窗口 | FOMO冲动时，强制等待24小时再做决定 | 避免冲动 |

**实战口诀**：管住手，不频繁。

---

### 3. 锚定效应 + 参考点依赖

**定义**：过度依赖第一个获取的数值（锚），而非客观技术位

**在交易中的表现**：
- 止损设在"入场价附近"而非技术支撑位
- 持仓目标变为"回到入场价就卖"
- 历史高点锚定：币价从高点跌50%就觉得"很便宜"

**规避策略**：

| 策略 | 操作 | 效果 |
|------|------|------|
| 技术面止损优先 | 止损位基于支撑/阻力，不基于成本 | 客观决策 |
| 固定风险比例 | 每笔交易最大亏损2% | 标准化风控 |
| 回本≠正确 | 每笔交易独立评估，不以"回本"为持仓理由 | 排除锚定 |
| 定期参考点回顾 | 每周末问：不持有现在会买吗？ | 排除入场价干扰 |

**实战口诀**：锚在技术位，不在成本价。

---

### 4. 羊群效应 + FOMO

**定义**：模仿他人行为，害怕错过上涨机会而追高

**在交易中的表现**：
- 看到社群喊单，立刻追入
- 某币连续暴涨3天，觉得"还会继续"，高位入场
- 主流媒体大幅报道时，往往已过最佳买点

**规避策略**：

| 策略 | 操作 | 效果 |
|------|------|------|
| 信号来源审核 | 只信经过验证的独立技术信号 | 排除喊单干扰 |
| 24小时冷静期 | FOMO冲动时，等待24小时再做决定 | 避免追高 |
| 只对策略负责 | 别人的盈利截图与己无关 | 排除社交压力 |
| 热度监控 | Google Trends等监控市场热度 | 热度极高时警惕 |
| 金字塔式建仓 | 不在高点一次性全仓买入 | 分摊成本 |

**实战口诀**：别人贪婪我警惕，别人恐惧我贪婪。

---

### 5. 确认偏差 + 选择性关注

**定义**：只看到支持自己观点的信息，忽略相反信号

**在交易中的表现**：
- 持有某币后，只关注利好，自动过滤利空
- 持仓亏损时，觉得"市场错了，不是我错了"
- 只看自己用的指标，忽略其他信号

**规避策略**：

| 策略 | 操作 | 效果 |
|------|------|------|
| 逆向思考 | 每笔持仓必须列出"为什么不持有"的2-3个理由 | 强制正反分析 |
| 信息多元化 | 主动阅读与自己观点相反的分析 | 打破信息茧房 |
| 决策日志 | 记录决策原因和依据，事后复盘 | 客观评估 |
| 预警信号清单 | 列出必须立即处理的5种危险信号 | 不依赖主观 |
| 错误优先级 | 非主观错误（API/技术）优先处理 | 系统化 |

**实战口诀**：逆向思考，正反都说。

## 二、交易日志模板

### 每次开仓前必须填写

```
## [日期时间] 开仓记录

### 基础信息
- 币种：______
- 方向：______（做多/做空）
- 入场价：______
- 止损价：______（必须基于技术位，不是入场价）
- 止盈价：______
- 仓位：______%
- 杠杆：______x

### 入场理由（必须基于技术分析）
- 缠论信号：______（分型/笔/中枢）
- 威科夫信号：______（吸筹/派发/Spring/SOS）
- 支撑阻力：______（关键价位）
- K线形态：______（锤子/吞没等）

### 情绪评估（1-5分）
- 开仓时情绪：______
- 是否有FOMO冲动：______（是/否）

### 计划遵守
- [ ] 止损预先设定
- [ ] 未在盘中临时更改计划
- [ ] 仓位符合规则

### 信号来源
- [ ] 技术信号，不是喊单
- [ ] 经过24小时冷静期（如果是FOMO驱动）
```

### 每次平仓后必须填写

```
## [日期时间] 平仓记录

### 交易结果
- 平仓价：______
- 盈亏：______%（或$______）

### 复盘
- 做对的：______
- 做错的：______
- 触发了哪种偏差：
  - [ ] 损失厌恶/处置效应
  - [ ] 过度自信/过度交易
  - [ ] 锚定效应
  - [ ] 羊群/FOMO
  - [ ] 确认偏差
- 下次改进：______
```

## 三、每日/每周复盘检查清单

### 每日（收盘后）

| # | 检查项 | 关联偏差 |
|---|--------|---------|
| 1 | 止损是否基于技术位而非入场价？ | 锚定效应 |
| 2 | 有没有因为"亏了这么多不想平"而持有？ | 损失厌恶 |
| 3 | 有没有在22:00后开新仓（冲动交易）？ | 过度自信 |
| 4 | 有没有超过24小时未解决的技术问题？ | 确认偏差 |
| 5 | 持仓理由中，正方和反方各几条？ | 确认偏差 |

### 每周（周日22:00）

| # | 检查项 | 关联偏差 |
|---|--------|---------|
| 1 | 本周交易频率 vs 胜率，是否过度交易？ | 过度自信 |
| 2 | 亏损最大的那笔，触发的是哪种偏差？ | 综合 |
| 3 | 本周有没有追社群喊单的操作？ | 羊群/FOMO |
| 4 | 策略手册有没有需要更新的地方？ | 过度自信 |
| 5 | 错误日志有没有超过24小时未解决的？ | 确认偏差 |

## 四、认知偏差触发链与打破点

```
损失厌恶 + 处置效应 → 不愿认赔，越亏越持
           ↓
锚定效应（入场价）→ 止损无限后移
           ↓
确认偏差 → 只看到支持持仓的信息
           ↓
过度自信 → 高杠杆 + 频繁交易
           ↓
羊群/FOMO → 高位追入 + 仓位失控
           ↓
结果：爆仓
```

**打破循环的关键**：在任何一环介入即可。最有效的打破点：

1. **开仓前设置硬止损**（打破锚定+处置）
2. **交易前等待24小时冷静期**（打破FOMO+过度自信）
3. **每周强制复盘**（打破确认偏差）

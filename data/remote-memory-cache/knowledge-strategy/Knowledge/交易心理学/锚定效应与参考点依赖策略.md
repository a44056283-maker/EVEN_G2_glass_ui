## 一，核心定义

**锚定效应（Anchoring）**：在做决策时过度依赖第一个获取的数值（锚），即使该数值与决策无关。

**参考点依赖**：盈亏判断依赖主观参考点（通常是入场价），而非客观市场结构。

---

## 二，交易中的典型症状

| 症状 | 表现 | 危害 |
|------|------|------|
| 入场价锚定 | 止损设在入场价附近而非技术位 | 止损失效 |
| 保本锚定 | "回到入场价就卖" | 错过大行情 |
| 历史高低点锚定 | 觉得从高点跌50%就是便宜 | 接飞刀 |
| 整数关口锚定 | 账户$10,000成为"保本线" | 恐慌性平仓 |

---

## 三，我的案例

**案例：止损被入场价牵引**
- 在$42,500做多SOL，快速跌至$39,000
- 将止损从$38,500下移至$37,000，理由是"技术面没破"
- 实际：被入场价锚定干扰，而非基于当前市场分析
- 结果：价格续跌至$32,000，亏损扩大至25%

## 四，系统化规避方法

### 1. 止损只基于技术位

```
规则：止损 = 当前支撑位 - ATR缓冲
不是：止损 = 入场价 - X%
```

### 2. 用ATR替代入场价作为锚

```python
def get_atr_stop(entry_price, atr, atr_mult=2.0, direction='long'):
    """
    ATR动态止损
    入场价仅用于计算距离，决策完全基于波动率
    """
    if direction == 'long':
        return entry_price - atr_mult * atr
    else:
        return entry_price + atr_mult * atr
```

### 3. "反直觉警告"触发条件

当出现以下念头时，强制复盘：
- "已经跌了X%了"
- "涨了X%了"
- "回到成本价就卖"

自问：我是在用"接下来会怎么走"决策，还是在用"和入场价差多少"决策？

### 4. 定期归零框架

每周问自己：如果我现在空仓，会在这个点位入场吗？
排除入场价干扰，纯粹评估当前市场结构。

> 止损是保护，不是"回本工具"。

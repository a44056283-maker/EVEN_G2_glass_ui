# 过度自信与过度交易 — 太子专属策略

> 版本：v1.0 | 更新：2026-04-03
> 来源：兵部overconfidence.md
> 定位：识别和规避过度自信导致的过度交易

---

## 一，核心定义

**过度自信（Overconfidence）**：系统性高估自己的知识水平、预测能力和交易技能，同时低估风险。

**过度交易（Overtrading）**：交易频率超过理性最优，Barber & Odean（2000）研究显示：过度自信的男性投资者年均收益比市场低2.65%。

---

## 二，交易中的典型症状

| 症状 | 表现 | 危害 |
|------|------|------|
| 频繁择时 | 相信能精准判断入场出场时机 | 手续费损耗 |
| 仓位放大 | 连盈后仓位从10%→50% | 一次亏损归零 |
| 高杠杆叠加 | 同时持有多个10x+仓位 | 极端风险 |
| "感觉良好" | 连续盈利后觉得"掌握了规律" | 过度优化 |

---

## 三，我的案例

**观察：止损被扫后仓位加倍**
- 设置20x杠杆做多，被扫后立即反手20x做空
- 连续被打脸2-3次，单日亏损超过30%
- 心理机制：每次止损后归咎于"就差一点点"，本质是归因偏差
- 正确做法：止损是市场噪音还是策略失效？先分析再交易

## 四，系统化规避方法

### 1. 交易频率追踪表

```python
# 每周检查
def check_trading_frequency(weekly_trades, max_weekly=5):
    """
    每周交易上限：5笔
    超出则强制冻结3天
    """
    if weekly_trades > max_weekly:
        return {"action": "FREEZE", "days": 3, "reason": "过度交易"}
    return {"action": "OK"}
```

### 2. 杠杆递减规则

| 连续亏损 | 杠杆上限 | 恢复条件 |
|---------|---------|---------|
| 2笔 | 50%正常 | 1笔盈利 |
| 3笔 | 25%正常 | 连续2笔盈利 |
| 止损触发 | 禁止开新仓 | 复盘后决定 |

### 3. 仓位上限硬规则

```
单品种仓位 ≤ 15%账户
所有合约仓位 ≤ 40%账户
永远保留 ≥ 60%救命资金
```

### 4. "专家装傻"自问

每次交易前必须回答：
- 这笔交易我判断正确的概率真的高于50%吗？
- 我可能漏掉了什么反向逻辑？
- 这个判断是基于分析还是"感觉"？

### 5. 信号与决策分离

```
信号产生 → 系统/策略（不是人脑）
执行 → 人工
盘中临时起意 → 禁止
```

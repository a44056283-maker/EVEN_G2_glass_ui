> 版本：v1.0 | 更新：2026-04-03

> 定位：识别FOMO，避免追高

---

## 一，核心定义

**羊群效应**：个体在信息不确定时，倾向于模仿大多数人的行为，认为群体选择包含了私有信息。

**FOMO（Fear of Missing Out）**：害怕错过大行情而追高买入。FOMO是羊群效应在社交媒体时代的人格化变体。

---

## 二，FOMO买入的典型阶段

```
第一阶段（酝酿）：少数早期建仓，社群升温，主流媒体未关注
     ↓
第二阶段（引爆）：KOL喊单/名人转发，价格快速上涨
     ↓
第三阶段（FOMO高潮）：群里晒盈利截图，新人跑步入场
     ↓  ← 大多数人在这里接盘
第四阶段（崩塌）：价格见顶横盘，然后快速下跌
```

---

## 三，FOMO高潮的识别信号

| 信号 | 说明 |
|------|------|
| 叙事极端化 | "这次不一样"、"改变世界" |
| 讨论热度与基本面脱节 | 产品没进展但币价暴涨 |
| 新人开始编"内部消息" | 没有实质信息的买入理由 |
| KOL集中喊单 | 同一币种短时间内被多个KOL推荐 |

## 五，系统化规避方法

### 1. 买入规则写在FOMO之前

```
在未看到价格变动之前，写下：
- 什么条件下会买入
- 仓位多大
- 止损在哪里

然后严格执行，不因Twitter讨论而改变
```

### 2. 24小时冷静期

```
规则：看到热门币种消息后，等待24小时再做决定
FOMO特点：让人觉得"必须现在行动"
冷静期设计：针对这一点
```

### 3. 信息源黑白名单

| 优先级 | 来源 | 权重 |
|--------|------|------|
| 高 | 财务数据、链上数据 | 100% |
| 中 | 核心社区讨论 | 50% |
| 低 | KOL推文 | 20% |
| 禁止 | 喊单群、付费vip | 0% |

### 4. 反向止盈机制

```
持仓浮盈超过50%时，自动止盈25%
即使FOMO继续推高价格，仍保留仓位参与行情
同时在高位锁定部分利润
```

### 5. 极端化观察

当出现以下叙事时，往往是反向信号：
- "这次不一样"
- "突破天际"
- "再也没有机会了"

## 一，核心定义

**近期偏差（Recency Bias）**：过度权重最近的信息和经验，系统性低估或忽视更早的历史数据。

**通俗解释**：大脑天然"重近轻远"——最近赚的钱感觉像实力，最近亏的钱感觉像运气。

---

## 二，交易中的典型症状

| 症状 | 表现 | 危害 |
|------|------|------|
| 圣杯幻觉 | 最近连续盈利后仓位放大 | 策略过拟合 |
| 选择性遗忘 | 清晰记得最近3笔，模糊早期亏损 | 自我评估失真 |
| 牛市思维惯性 | 用最近市场状态判断未来 | 熊市抄底被埋 |

## 三，我的案例

**观察：Meme币周期**
- 将"做空某币赚钱"归纳为"我擅长做空Meme币"
- 在后续Meme牛市启动时继续做空
- 结果：遭遇50-200%爆仓损失

## 四，系统化规避方法

### 1. 滚动绩效记录

```
强制记录：3个月/6个月/12个月收益率
而非仅记录单日/单周盈亏
当短期收益率显著高于长期 → 近期偏差预警
```

### 2. 决策-结果分离日志

```
决策记录：买入理由、仓位、止损
结果记录：盈亏金额
两者分开存储，不在同一条目显示
复盘时只看决策记录
```

### 3. 策略冷静期

```
当市场环境发生重大变化（牛市→熊市）
强制进入冷静期
重新评估当前策略是否适用
```

## 核心概念

### 行为金融学实盘应用框架

**核心命题**: 每一次交易失误，背后都有一条清晰的偏差链条。找到这条链条，就找到了规则干预的切入点。

## 一、偏差-决策绑定表（4种核心偏差 × 典型交易场景）

### 1. 过度自信 → 具体决策路径

| 场景 | 偏差驱动的决策 | 危险信号 | 规则干预点 |
|------|---------------|---------|-----------|
| 看到利好立刻开仓 | 高估自己解读信息的能力 | 开仓前未列利空清单 | 强制等待4小时 |
| 连续盈利后加仓 | 高估策略有效期 | 盈亏比恶化但不减仓 | 凯利公式上限 |
| 多个策略同时跑 | 高估自己能同时管理 | 监控不过来漏掉强平 | 持仓数硬上限3个 |
| 亏损后立即入场 | 高估自己能"扳回来" | 无冷静期，报复性交易 | 强制48小时冻结 |

**今日核心理解**: 过度自信不是"骄傲"，而是"校准失败"——把自己的主观信心当成了客观准确率。实盘中"我觉得会涨"和"涨的概率>60%"是两件事。

---

### 2. 损失厌恶 → 具体决策路径

| 场景 | 偏差驱动的决策 | 危险信号 | 规则干预点 |
|------|---------------|---------|-----------|
| 止损触发前撤销 | 不愿意实现亏损 | 同一仓位撤销止损>1次 | 撤销需二次确认 |
| 盈利先出亏损留 | 持有亏损卖出盈利 | 亏损持仓时间>盈利持仓2倍 | 月度处置效应评分 |
| 回本就跑 | 损失痛苦感是盈利快感2倍 | 止盈目标=入场价 | 盈亏比≥2:1硬规则 |
| 账户浮亏不看 | "不看了就不存在" | 连续3天未登录账户 | 每日强提示 |

**今日核心理解**: 损失厌恶在脑科学层面是杏仁核劫持——爆仓的痛苦等同于身体受伤。靠意志力对抗杏仁核必然失败，必须用规则替代决策（预设止损单）。

---

### 3. 锚定效应 → 具体决策路径

| 场景 | 偏差驱动的决策 | 危险信号 | 规则干预点 |
|------|---------------|---------|-----------|
| 止损设在场内价附近 | 第一信息锚定（成本价）| 止损<2%区间 | 止损必须基于ATR技术位 |
| "回到本就卖" | 入场价成为参考点 | 持仓理由写"回本" | 回本理由=禁止开仓信号 |
| 价格从高点跌50%觉得便宜 | 历史高点锚定 | 未评估基本面/趋势 | 低位≠买入信号清单 |
| 卖出后价格继续涨，追回 | 新参考点=卖出价 | 卖后48小时内追入 | 追入需二次验证 |

**今日核心理解**: 锚定效应的本质是"参考点不是客观的，而是主观被第一个信息占领了"。打破锚定的方法：每次决策前先问"如果我没有持仓，现在会买吗？"——不是问"现在要不要卖"，因为后者默认锚定在持仓状态。

---

### 4. 确认偏差 → 具体决策路径

| 场景 | 偏差驱动的决策 | 危险信号 | 规则干预点 |
|------|---------------|---------|-----------|
| 只看支持持仓的帖子 | 选择性关注 | 持仓期间利空文章未读 | 入场前必须写"反向理由" |
| 忽略预警信号 | 选择性忽视 | 保证金率<120%未处理 | 预警信号清单≥5条 |
| 持仓期间频繁刷社群 | 寻找确认而非信息 | 刷社群>10分钟/天 | 社媒使用时间记录 |
| 否认错误，怪市场 | 自我归因偏差 | 亏损归咎"庄家/消息" | 月度归因复盘表 |

**今日核心理解**: 确认偏差最危险的形式是"我觉得我已经考虑周全了"——这句话本身就是确认偏差的体现。破解方法是强制要求"反向理由"，写不出来=禁止开仓。

## 二、偏差量化 → 实盘决策映射

### 已知量化指标（来自04-09笔记）

| 偏差 | 量化指标 | 计算公式 |
|------|---------|---------|
| 过度自信 | 预测偏差 | \|预测涨幅 - 实际涨幅\| / 实际涨幅 |
| 损失厌恶 | 盈亏比 | 平均盈利 / 平均亏损 |
| 锚定效应 | 止损集中度 | 止损位在入场价±2%内的比例 |
| 确认偏差 | 无逆向理由比例 | 入场前未写反向理由的比例 |

### 决策映射规则

```
IF 预测偏差 > 50%  →  下一周交易频率减半
IF 盈亏比 < 1.5     →  审查止盈规则是否被情绪干扰
IF 止损集中度 > 40% →  强制要求技术止损位截图存档
IF 无逆向理由 > 30% →  禁止开仓1周（需书面申请解禁）
```

## 六、核心要点（3个）

1. **偏差链条是交易失误的DNA**: 过度自信→高杠杆，损失厌恶→不止损，锚定效应→止损后移，确认偏差→忽略预警——每笔错误交易都能在偏差链条上找到对应节点。知道偏差链条，才能找到干预点。

2. **层次2比层次1重要10倍**: 识别偏差是事后复盘，预测偏差是事前防御。"我能预判自己会犯什么错"才能真正改变行为，而不是每次亏损后说"我又犯了过度自信的毛病"。

3. **规则替代决策是唯一有效解**: 杏仁核劫持（损失厌恶）不能靠意志力克服，只能用预设止损单替代情绪决策。规则替代的关键是：规则必须在情绪升起之前就已经存在并生效。

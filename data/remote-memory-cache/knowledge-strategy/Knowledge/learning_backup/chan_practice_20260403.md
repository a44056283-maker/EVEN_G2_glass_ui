## 理论复习：分型三要素

### 顶分型结构
```
K线1: 高点 = H1
K线2: 高点 = H2  ← 中间最高
K线3: 高点 = H3

条件：H2 > H1 且 H2 > H3
意义：多方力量在K线2达到顶峰后衰竭
```

### 底分型结构
```
K线1: 低点 = L1
K线2: 低点 = L2  ← 中间最低
K线3: 低点 = L3

条件：L2 < L1 且 L2 < L3
意义：空方力量在K线2达到顶峰后衰竭
```

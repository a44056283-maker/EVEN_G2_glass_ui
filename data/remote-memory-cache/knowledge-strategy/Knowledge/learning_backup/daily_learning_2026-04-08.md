## 一、绩效归因的核心目的

**归因 = 解释收益来源**

不问"赚了多少钱"，问"钱从哪里来，是策略的本事还是市场的恩赐"。

---

## 二、三层归因框架（币圈实战版）

### 第一层：Beta vs Alpha
```
总收益 = Beta收益 + Alpha收益
```

- **Beta收益**：市场上涨带动的收益，顺势而行
- **Alpha收益**：策略独立于市场的超额收益，真实本事

**判断标准**：
- Beta > 1：放大波动，牛市赚更多，熊市也亏更多
- Alpha > 0：策略有真实价值，与大盘无关

### 第二层：Brinson归因（适合多币种组合）

将超额收益拆解为三个效应：

| 效应 | 含义 | 公式 |
|------|------|------|
| **配置效应** | 偏离基准行业权重带来的收益 | Σ(实际权重 - 基准权重) × 基准收益 |
| **选股效应** | 每个行业内部选币带来的超额 | Σ实际权重 × (实际收益 - 基准收益) |
| **交互效应** | 配置×选股的交叉影响 | 配置与选股共同作用的部分 |

**币圈简化**：用币种替代行业，用市场指数替代基准组合

### 第三层：归因质量检验

| 问题 | 症状 | 解决方案 |
|------|------|----------|
| Alpha不稳定 | 每月Alpha波动大 | 增加样本或检查过拟合 |
| Beta漂移 | 策略对市场敏感度变化 | 检查策略是否失效 |
| 高交互效应 | 配置和选股难以区分 | 简化策略结构 |

## 三、信息比率（Information Ratio）

```
IR = Alpha / 跟踪误差
   = (策略收益 - 基准收益) / 策略与基准收益差值的标准差
```

**含义**：每单位主动风险带来的超额收益

**评级标准**（业界常用）：
- IR < 0.5：较差
- IR 0.5-1.0：及格
- IR > 1.0：优秀
- IR > 2.0：顶级

**与夏普比率的区别**：
- 夏普：超额收益（相对无风险利率）/ 总风险
- IR：超额收益（相对基准）/ 主动风险

## 四、归因分析的实操步骤

```python
def performance_attribution(strategy_returns, market_returns, benchmark_returns=None):
    """
    币圈绩效归因完整流程
    """
    import numpy as np

    # Step 1: 计算基本统计量
    total_return = np.prod(1 + np.array(strategy_returns)) - 1
    market_return = np.mean(market_returns)

    # Step 2: Beta-Alpha分解
    cov = np.cov(strategy_returns, market_returns)[0][1]
    market_var = np.var(market_returns)
    beta = cov / market_var if market_var != 0 else 0
    alpha = np.mean(strategy_returns) - beta * market_return

    # Step 3: 跟踪误差
    tracking_error = np.std(np.array(strategy_returns) - np.array(market_returns))
    ir = alpha / tracking_error if tracking_error != 0 else 0

    # Step 4: 归因总结
    result = {
        '总收益': f'{total_return:.2%}',
        'Beta': round(beta, 2),
        'Alpha(日)': f'{alpha:.4f}',
        'Alpha(年化)': f'{(alpha * 365):.2%}',
        '跟踪误差(日)': f'{tracking_error:.4f}',
        '信息比率(年化)': f'{ir * np.sqrt(365):.2f}'
    }

    return result
```

## 五、常见归因陷阱

### 1. 归因时间错配
用月度收益归因到每日信号——信号在月内早就平仓了
**解决**：归因周期必须与决策周期一致

### 2. 多因子共线性
多个因子同时暴露，分不清谁贡献了多少
**解决**：逐步回归或正交化处理

### 3. 忽略交易成本
归因出的Alpha可能全被手续费吃掉
**解决**：归因前先扣除所有成本

### 4. 基准选择不当
用不相关的指数做基准，Beta计算无意义
**解决**：选择与策略相关性 > 0.5 的基准

### 第4周核心洞察（3个）

1. **回测的三个死亡陷阱**：过拟合（参数在历史数据上挖出来）、前视偏差（偷看未来数据）、生存者偏差（只用活着的数据测试）。任何一个都让回测结果完全不可信。

2. **绩效指标各有分工**：夏普看综合风险调整收益，索提诺只看下行风险，最大回撤看的是复利引擎的损伤程度。三个指标合在一起才能全面评估一个策略。

3. **归因是策略的照妖镜**：高收益可能是高Beta带来的，看起来很强的策略可能只是赌对了市场方向。Alpha稳定、IR > 1才是真正有本事的策略。

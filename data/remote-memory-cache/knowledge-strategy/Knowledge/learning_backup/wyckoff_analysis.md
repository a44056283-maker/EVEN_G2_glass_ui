## 一、核心哲学：供需决定价格

威科夫方法的核心：**价格由供需关系决定**。

- 供应占优 → 价格下跌
- 需求占优 → 价格上涨
- 成交量是供需强度的真实反映

### 威科夫三定律

| 定律 | 内容 | 实战意义 |
|------|------|----------|
| **第一定律** | 供需决定价格 | 上涨必须放量，下跌可缩量 |
| **第二定律** | 因果关系 | 趋势反转前必有横盘（努力/结果） |
| **第三定律** | 投入产出原则 | 震荡区间越大，未来趋势越强 |

## 二、吸筹过程（主力建仓）

### 吸筹阶段特征（Spring + SOS + LPS）

```
         ┌─────────────────────────────────────┐
         │           吸筹区间（区间交易）         │
         │                                     │
    ─────┤  PS（初步支撑）  ──────────────────│ ← 主力开始买入
         │       ↓                             │
         │  Spring（弹簧效应）                  │ ← 突破支撑后快速拉回
         │       ↓                             │   说明供应被吸收
         │  SOS（强势信号）  ────────────────│ ← 价格反弹，量增
         │       ↓                             │
         │  LPS（最后支撑）  ─────────────────│ ← 回踩，不破前低
         │       ↓                             │
         └─────────────────────────────────────┘
                   ↑ 突破区间上沿 = 买入信号
```

### 吸筹关键点

| 术语 | 含义 | 信号意义 |
|------|------|----------|
| **PS (Preliminary Support)** | 初步支撑 | 聪明钱开始试探性买入 |
| **SC (Selling Climax)** | 卖出高潮 | 最后一波恐慌抛售，通常是区间最低点 |
| **Spring** | 弹簧效应 | 跌破支撑后快速拉回 = 供应被吸收，主力吸筹 |
| **SOS (Sign of Strength)** | 强势信号 | 反弹时量增价升 = 需求强于供应 |
| **LPS (Last Point of Support)** | 最后支撑点 | 回调不破前低 = 二次确认需求强势 |
| **BU (Back Up)** | 二次测试 | 突破后回踩区间上沿，不破 = 有效突破 |

## 三、派发过程（主力出货）

### 派发阶段特征（Upthrust + LPSY + SOW）

```
         ┌─────────────────────────────────────┐
         │           派发区间（区间交易）         │
         │                                     │
         │────────────────→  PSY（初步供应）────┤ ← 主力开始出货
         │       ↓                             │
         │  UT（突破效应）                      │ ← 突破阻力后快速回落
         │       ↓                             │   说明需求被消耗
         │  LPSY（最后供应点） ───────────────│ ← 反弹受压，无力再创新高
         │       ↓                             │
         │  SOW（弱势信号）  ────────────────│ ← 下跌时量增 = 供应强于需求
         └─────────────────────────────────────┘
                   ↓ 跌破区间下沿 = 卖出信号
```

### 派发关键点

| 术语 | 含义 | 信号意义 |
|------|------|----------|
| **PSY (Preliminary Supply)** | 初步供应 | 主力开始试探性卖出 |
| **UT (Upthrust)** | 突破效应 | 突破阻力后快速回落 = 需求被消耗，主力出货 |
| **LPSY (Last Point of Supply)** | 最后供应点 | 反弹受压，无法再创新高 |
| **SOW (Sign of Weakness)** | 弱势信号 | 下跌时量增 = 供应强于需求 |
| **BC (Buying Climax)** | 买入高潮 | 最后一波狂热追高，通常是区间最高点 |

## 四、量价关系四大情形

| 情形 | 量 | 价 | 解读 |
|------|----|----|------|
| **供不应求** | 放量 | 上涨 | 健康上涨，需求主导 |
| **供过于求** | 放量 | 下跌 | 健康下跌，供应主导 |
| **虚涨** | 缩量 | 上涨 | 主力拉高出货预警 |
| **虚跌** | 缩量 | 下跌 | 主力压低吸筹预警 |

### 实战口诀
- **上涨必须放量**：无量上涨 = 主力未参与 = 不可持续
- **下跌可以缩量**：缩量下跌 = 供应枯竭 = 可能反转
- **突破需要放量**：缩量突破 = 假突破概率高
- **震荡缩量**：等待方向的平静期

## 五、威科夫综合方法（Composite Operator）

威科夫认为：每个股票都有一只"看不见的手"（Composite Operator，主力运营商）在操纵。

### 识别主力意图的4步法

```
Step 1: 确定区间（Trading Range）
        → 横盘震荡的上下边界是什么？

Step 2: 判断区间性质（是吸筹还是派发？）
        → 看区间内低点是否抬高（吸筹）vs 高点降低（派发）
        → 看成交量分布（低位放量=吸筹，高位放量=派发）

Step 3: 找触发点（Schub）
        → 向上突破上沿 = 买入信号
        → 向下突破下沿 = 卖出信号

Step 4: 止损设置
        → 买入后止损放在区间下沿下方
        → 卖出后止损放在区间上沿上方
```

## 六、吸筹 vs 派发的8个鉴别点

| 特征 | 吸筹（建仓） | 派发（出货） |
|------|-------------|-------------|
| 区间低点 | 逐步抬高 | 逐步下降 |
| 区间高点 | 相对固定 | 逐步降低 |
| 成交量分布 | 低位放量 | 高位放量 |
| 弹簧效应(Spring) | 跌破支撑后拉回 | 跌破后无力反弹 |
| 突破方向 | 向上突破 | 向下突破 |
| 突破量能 | 放量突破 | 缩量突破或突放巨量 |
| 回调深度 | 浅回调（不破前低） | 深回调（破前高） |
| 持仓变化 | 机构持仓增加 | 机构持仓减少 |

## 七、威科夫与缠论的异同

| 维度 | 缠论 | 威科夫 |
|------|------|--------|
| **核心** | 结构（几何） |供需（动力学） |
| **最小单元** | 分型 | 分型（类似） |
| **趋势定义** | 笔/线段/中枢 | 吸筹/派发区间 |
| **买卖点** | 一买/二买/三买 | Spring/LPS/SOS |
| **量价关系** | 不重点关注 | 核心关注 |
| **主力意图** | 不直接分析 | 重点分析 |

### 互补性
- 缠论擅长**几何结构**：精确的笔/线段/中枢定位
- 威科夫擅长**量价关系**：主力意图识别
- **最佳用法**：先用缠论定位结构，再用威科夫确认买卖点

## 八、实战应用步骤

### Step 1: 画区间
```
找到横盘震荡的上轨和下轨
→ 标注区间（TR, Trading Range）
```

### Step 2: 判读性质
```
数轴高低点：
  低点抬高 → 吸筹嫌疑
  高点降低 → 派发嫌疑

量能分布：
  低位放量 → 吸筹
  高位放量 → 派发
```

### Step 3: 找信号
```
吸筹信号：Spring后SOS = 买入
派发信号：UT后LPSY = 卖出
```

### Step 4: 止损
```
买入：止损在区间下沿下方1-2%
卖出：止损在区间上沿上方1-2%
```

## 十、核心要点（3条）

1. **供需决定价格**：放量方向即主力方向，上涨无量=虚涨不可持续
2. **吸筹看Spring**：跌破支撑快速拉回=主力吸筹=买入信号；派发看UT：突破阻力快速回落=主力出货=卖出信号
3. **威科夫+缠论互补**：缠论定结构（精确买点），威科夫判意图（确认主力方向）

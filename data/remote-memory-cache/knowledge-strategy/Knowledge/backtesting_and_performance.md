# 回测与绩效评估

> 学习日期：2026-04-06 | 所属周：第4周

---

## 一、三大回测陷阱

### 1. 过拟合（Overfitting）

**定义：** 策略过度适应历史数据的特定噪声，而非捕捉真实的市场规律。

**常见形式：**
- 参数优化过细（参数孤岛效应）
- 规则过于复杂（维数灾难）
- 使用过多指标/过滤条件
- 在小样本数据上反复调参

**识别方法：**
- 样本外夏普比率显著低于样本内
- 参数敏感性分析：微调参数导致绩效大幅波动
- 步进式分析：逐步增加参数，观察绩效曲线

**应对方法：**
- 留出足够长的"冷启动期"（至少1年）
- 蒙特卡洛模拟：随机打乱历史数据重新测试
- 简化参数：宁可欠拟合，也不要过拟合
- 约束参数数量：参数数 ≤ 样本数据点 / 10

**实盘警告：** 过拟合策略在实盘中胜率往往腰斩。

---

### 2. 前视偏差（Look-Ahead Bias）

**定义：** 回测中使用了在实际交易时无法获得的信息。

**常见形式：**
- 收盘价陷阱：盘中利用收盘价信号，而实际撮合无法以收盘价成交
- 财务报表前视：使用"宣告日"而非"实际生效日"的数据
- 未来函数：使用了回测区间内未来才出现的数据
- 非快照数据：使用当日收盘后的数据做当日决策

**典型案例：**
```
错误示范：if close > close[1] and close[1] > close[2]: buy()
         （实际撮合时，close[1]是昨日收盘，是历史数据）

正确做法：基于已关闭的K线做判断，用下一根K线开盘价执行
```

**应对方法：**
- 严格使用"bar内不可知"的数据
- 每次信号用下一根K线开盘价模拟成交
- 滑点建模：加入0.1%~0.3%滑点缓冲

---

### 3. 生存者偏差（Survivorship Bias）

**定义：** 只用当前存活的数据（股票池）做回测，忽略了历史上已经退市/崩盘的公司。

**后果：**
- 回测绩效被高估20%~40%
- 波动率被低估
- 极端收益事件被忽略

**典型案例：**
2000年互联网泡沫前纳入NASDAQ指数的股票，很多已退市。如果只测试"当前存在"的公司，会自动排除退市输家。

**应对方法：**
- 使用包含已退市股票的完整历史数据库（如 CSI、TickData）
- 每次调仓日，使用当时实际存在的股票池（_point-in-time data*）
- 交叉验证：用不同数据源对比

## 二、三大绩效指标

### 1. 夏普比率（Sharpe Ratio）

**公式：**
```
SR = (平均收益率 - 无风险利率) / 收益率标准差
```

**解读：**
- SR > 1：优秀
- SR > 2：卓越
- SR < 0.5：一般

**局限性：**
- 假设收益正态分布
- 不区分上行和下行波动

---

### 2. 索提诺比率（Sortino Ratio）

**公式：**
```
Sortino = (平均收益率 - 目标收益率) / 下行标准差
```

**改进：** 只惩罚下行波动，不惩罚上行波动，比夏普更反映真实风险。

---

### 3. 最大回撤（Maximum Drawdown）

**定义：** 从历史最高点到最低点的最大跌幅。

**为什么重要：**
- 心理承受力决定能否坚持策略
- 影响复利效应（回撤20%需要涨25%才能回本）
- 凯利公式中，最大回撤决定仓位的抗崩溃边界

**监控阈值建议：**
- 警告线：5%
- 硬止损：10%
- 策略废弃线：15%~20%

---

## 三、绩效归因分析

**目的：** 分解收益来源，理解策略的真实Alpha。

**分解框架：**
```
总收益 = Beta收益 + Alpha收益
       = 市场敞口收益 + 选股收益 + 择时收益 + 其他
```

**步骤：**
1. 剥离市场基准收益（Benchmark）
2. 计算选股Alpha（个股选择 vs 指数）
3. 计算择时贡献（仓位变化带来的超额收益）
4. 计算风险暴露（行业/风格偏离）

## 四、回测框架自检清单

| 检查项 | 标准 |
|--------|------|
| 数据完整性 | 包含已退市股票，使用point-in-time数据 |
| 前视偏差 | 所有信号基于已关闭K线 |
| 过拟合 | 参数调优不超过数据量的10%，样本外绩效不大幅下滑 |
| 滑点建模 | 至少0.1%单边滑点 |
| 交易成本 | 佣金+滑点+冲击成本总和 |
| 流动性 | 小市值股票有流动性折扣 |

---

## 五、核心要点（今日学习）

1. **三大陷阱**：过拟合（参数孤岛）、前视偏差（收盘价陷阱）、生存者偏差（只测存活股票）
2. **绩效三指标**：夏普（综合风险）、索提诺（只看下行的）、最大回撤（心理/复利边界）
3. **自检清单**：数据完整性→信号无前视→参数不过拟→成本充分→流动性考虑

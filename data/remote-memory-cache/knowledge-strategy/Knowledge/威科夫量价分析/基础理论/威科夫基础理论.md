## 一、威科夫哲学

### 1.1 核心思想

> "价格由供需关系决定。成交量告诉我们供需的天平倾向哪一方。"

| 现象 | 含义 |
|------|------|
| 价涨 + 量增 | 需求强劲，上涨健康 |
| 价涨 + 量缩 | 需求虚弱，可能是假突破 |
| 价跌 + 量增 | 供应强劲，下跌健康 |
| 价跌 + 量缩 | 供应枯竭，可能是见底信号 |

### 1.2 威科夫三定律

| 定律 | 内容 | 实战应用 |
|------|------|---------|
| **第一定律** | 供需决定价格 | 上涨需要需求主动，下跌需要供应主动 |
| **第二定律** | 因果关系 | 趋势反转前必有横盘整理（努力与结果） |
| **第三定律** | 投入产出 | 横盘区间越大，未来趋势越强 |

## 二、吸筹过程 — 主力建仓

### 2.1 吸筹阶段全貌

```
阶段1: 初始支撑 PS
         ↓
阶段2: 恐慌抛售 SC (卖出高潮)
         ↓
阶段3: 二次测试 UT (弹簧效应/弹簧)
         ↓
阶段4: 吸筹区间震荡
         ↓
阶段5: 突破前夕 LPS (最后支撑)
         ↓
阶段6: 突破区间上沿 SOW→SOS (ST)
```

### 2.2 各阶段详解

#### PS (Preliminary Support) 初步支撑

**含义**：聪明钱开始试探性买入，价格形成初步支撑

**特征**：
- 成交量开始放大
- 价格跌幅趋缓
- 但还不是最低点

**实战**：关注但不操作，等待确认

#### SC (Selling Climax) 卖出高潮

**含义**：最后一波恐慌抛售，通常是区间的最低点

**特征**：
- 成交量急剧放大
- 价格快速下跌后快速拉回
- 伴随恐慌情绪（新闻、社交媒体唱空）

**实战**：
- 🔴 不要在这里恐慌卖出
- 🟢 如果仓位重，可以减仓但不割肉

#### Spring 弹簧效应（关键信号！）

**含义**：价格短暂跌破支撑后快速拉回

**特征**：
- 价格跌破SC低点
- 但成交量没有放大（或缩小）
- 价格迅速拉回支撑上方

**实战意义**：
- 跌破支撑但无新供应 = 主力在吸筹
- 这是**最可靠的买入信号之一**

```
Spring示意图：
        支撑线 ──────────────
       /
SC ──╯                    ← 卖出高潮（大量抛售）
      ╲                   
       ╲________________
       /
Spring ╲                 ← 短暂跌破支撑
       /╲________________
      /
支撑线 ──────────────
```

#### SOS (Sign of Strength) 强势信号

**含义**：价格反弹时成交量放大，价格涨幅大于之前跌幅

**特征**：
- 成交量明显放大
- 收盘接近日内高点
- 涨幅 > 之前跌幅的61.8%

**实战**：SOS出现 = 需求压倒供应，看涨

#### LPS (Last Point of Support) 最后支撑点

**含义**：突破前最后一次回踩支撑

**特征**：
- 价格回调，但不低于前低
- 成交量萎缩
- 回调幅度越来越小

**实战**：LPS是较好的买入机会

## 三、派发过程 — 主力出货

### 3.1 派发阶段全貌

```
阶段1: 初始供应 PSY
         ↓
阶段2: 买入高潮 BC (买入高潮)
         ↓
阶段3: 二次测试 UTAD (突破效应)
         ↓
阶段4: 派发区间震荡
         ↓
阶段5: 跌破前夕 LPSY (最后供应点)
         ↓
阶段6: 跌破区间下沿 SOW (弱势信号)
```

### 3.2 各阶段详解

#### PSY (Preliminary Supply) 初步供应

**含义**：主力开始出货，价格形成初步阻力

**特征**：
- 成交量开始放大
- 价格涨幅趋缓
- 但还不是最高点

#### BC (Buying Climax) 买入高潮

**含义**：最后一波散户追高买入，主力趁机出货

**特征**：
- 成交量急剧放大
- 价格快速上涨后快速回落
- 伴随贪婪情绪（新闻、社交媒体唱多）

**实战**：
- 🔴 不要在这里追高买入
- 🟢 如果仓位轻，不要加仓

#### UT/UTAD (Upthrust) 突破效应

**含义**：价格短暂突破阻力后快速回落

**特征**：
- 价格突破前高
- 但成交量没有跟随放大
- 价格迅速回落

**实战意义**：
- 突破阻力但无新需求 = 主力在出货
- 这是**最可靠的卖出信号之一**

```
UT示意图：
              阻力线 ──────────────
             ╱
BC ─────────╯                    ← 买入高潮（大量买入）
           ╱                     
          ╱                      
         ╱__________________╱
                             ╲
UTAD ────╲                   ← 短暂突破阻力
         ╱╲__________________
        ╱
阻力线 ──────────────
```

#### SOW (Sign of Weakness) 弱势信号

**含义**：下跌时成交量放大，价格跌幅明显

**特征**：
- 成交量明显放大
- 收盘接近日内低点
- 跌幅 > 之前涨幅的61.8%

**实战**：SOW出现 = 供应压倒需求，看跌

## 四、区间交易 — 核心战场

### 4.1 区间定义

区间 = 价格在 **支撑位** 和 **阻力位** 之间震荡的市场状态

| 术语 | 含义 |
|------|------|
| 支撑位 | 买方集中的价格区域 |
| 阻力位 | 卖方集中的价格区域 |
| 测试 | 价格接近支撑/阻力位时的反应 |

### 4.2 区间交易策略

#### 在吸筹区间交易

```
策略：低买高卖

买入信号：
1. 价格触及支撑位
2. 成交量萎缩（供应不足）
3. 出现反转K线（锤子、吞没等）

卖出信号：
1. 价格触及阻力位
2. 成交量放大（需求不足）
3. 出现反转K线
```

#### 在派发区间交易

```
策略：高卖低买（做空）

做空信号：
1. 价格触及阻力位
2. 成交量放大（供应涌入）
3. 出现反转K线

平仓信号：
1. 价格触及支撑位
2. 成交量萎缩
3. 出现反转K线
```

## 五、量价关系实战

### 5.1 八种基本量价关系

| 编号 | 价格 | 成交量 | 含义 | 操作 |
|------|------|-------|------|------|
| 1 | 涨 | 增 | 需求主动 | 持有/买入 |
| 2 | 涨 | 缩 | 需求虚弱 | 警惕/卖出 |
| 3 | 跌 | 增 | 供应主动 | 警惕/卖出 |
| 4 | 跌 | 缩 | 供应枯竭 | 关注/买入 |
| 5 | 震荡 | 增 | 主力博弈 | 观望 |
| 6 | 震荡 | 缩 | 观望情绪 | 等待突破 |
| 7 | 突破 | 增 | 真突破 | 顺势操作 |
| 8 | 突破 | 缩 | 假突破 | 反向操作 |

### 5.2 量价背离

| 类型 | 现象 | 信号 |
|------|------|------|
| 顶背离 | 价格上涨但成交量萎缩 | 看跌 |
| 底背离 | 价格下跌但成交量萎缩 | 看涨 |
| 放量滞涨 | 价格创新高但成交量未跟上 | 看跌 |
| 缩量止跌 | 价格创新低但成交量未放大 | 看涨 |

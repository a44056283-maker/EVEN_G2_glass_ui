## 一、威科夫哲学

### 1.1 核心思想

> "价格由供需关系决定。成交量告诉我们供需的天平倾向哪一方。"

| 现象 | 含义 |
|------|------|
| 价涨 + 量增 | 需求强劲，上涨健康 |
| 价涨 + 量缩 | 需求虚弱，可能是假突破 |
| 价跌 + 量增 | 供应强劲，下跌健康 |
| 价跌 + 量缩 | 供应枯竭，可能是见底信号 |

### 1.2 威科夫三定律

| 定律 | 内容 | 实战应用 |
|------|------|---------|
| **第一定律** | 供需决定价格 | 上涨需要需求主动，下跌需要供应主动 |
| **第二定律** | 因果关系 | 趋势反转前必有横盘整理（努力与结果） |
| **第三定律** | 投入产出 | 横盘区间越大，未来趋势越强 |

## 二、吸筹过程 — 主力建仓

### 2.1 吸筹阶段全貌

```
阶段1: 初始支撑 PS
         ↓
阶段2: 恐慌抛售 SC (卖出高潮)
         ↓
阶段3: 二次测试 UT (弹簧效应/弹簧)
         ↓
阶段4: 吸筹区间震荡
         ↓
阶段5: 突破前夕 LPS (最后支撑)
         ↓
阶段6: 突破区间上沿 SOW→SOS (ST)
```

### 2.2 各阶段详解

#### PS (Preliminary Support) 初步支撑

**含义**：聪明钱开始试探性买入，价格形成初步支撑

**特征**：
- 成交量开始放大
- 价格跌幅趋缓
- 但还不是最低点

**实战**：关注但不操作，等待确认

#### SC (Selling Climax) 卖出高潮

**含义**：最后一波恐慌抛售，通常是区间的最低点

**特征**：
- 成交量急剧放大
- 价格快速下跌后快速拉回
- 伴随恐慌情绪（新闻、社交媒体唱空）

**实战**：
- 🔴 不要在这里恐慌卖出
- 🟢 如果仓位重，可以减仓但不割肉

#### Spring 弹簧效应（关键信号！）

**含义**：价格短暂跌破支撑后快速拉回

**特征**：
- 价格跌破SC低点
- 但成交量没有放大（或缩小）
- 价格迅速拉回支撑上方

**实战意义**：
- 跌破支撑但无新供应 = 主力在吸筹
- 这是**最可靠的买入信号之一**

# 每日学习采集降噪铁律

## 一、最高目标

每日采集和学习不是为了堆知识,而是为了形成可验证的交易改进假设。

优先级:

1. 减少噪音下单。
2. 提高 V6.5 入场方向准确率。
3. 防止旧规则污染。
4. 提炼可进入 L5 影子实验室的候选规则。
5. 优化天眼/出山解释逻辑,但不得让 AI 越权开仓或平仓。

## 二、每日只采集五类内容

### 1. 合约多空策略

必须同时支持做多和做空。只做多、现货专用、网格现货套利类内容默认降权。

### 2. 资金流/OI/爆仓/盘口

重点采集:

- M1 资金流。
- OI 扩张/收缩。
- 多空爆仓方向。
- 盘口买卖倾斜。
- 点差和滑点风险。

### 3. 支撑压力与假突破识别

重点采集:

- 压力做多风险。
- 支撑做空风险。
- 无量突破。
- 缩量回踩。
- Spring / UT / SOW / SOS。

### 4. 仓位管理和风控

重点采集:

- 5x-10x 合约杠杆风险。
- 量能强弱对应仓位。
- ATR波动异常。
- 过早出场和过度止损风险。

### 5. 历史亏损单归因与噪音过滤

重点采集:

- 反向开单。
- 假突破。
- 低质量放量。
- M1-M5冲突。
- AI过早平仓。

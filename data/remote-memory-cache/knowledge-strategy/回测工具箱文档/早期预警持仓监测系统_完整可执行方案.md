```
## 决策规则

### 评分计算
- 方向错误（5信号，满足≥3条触发，每条20分）：
  1. RSI极端区+浮亏>1%（做多RSI>70，做空RSI<30）
  2. RSI方向背离（做多RSI<50，做空RSI>50）
  3. 跌破支撑>1% 或 突破阻力>1%
  4. 连续3根K线量比<1.0
  5. 铁律违反（做多不在支撑，做空不在压力）

- 过早入场（4信号，满足≥2条触发，每条25分）：
  1. 入场距S/R>3%（中间地带）
  2. 入场量比<2.5x
  3. R3引擎判定过早
  4. 入场RSI非极端值

### 分级轮次门控
- 评分<40（🔴高危）：1轮触发，立即执行
- 评分40-59（🟠中高危）：2轮触发
- 评分≥60（🟡中风险）：3轮触发

### 干预动作
| 评分 | 类型 | 动作 |
|------|------|------|
| 60-79 | 过早入场 | observe（保持观察） |
| 40-59 | 过早入场 | observe + 监控补仓线 |
| <40 | 过早入场 | half_close + 补仓线设置 |
| 50-79 | 方向错误 | half_close |
| <50 | 方向错误 | full_close（全平） |
| <40 | 方向错误+多重违反 | full_close + 标记高危 |

### 补仓参数
- ATR(14)：{atr_14}
- L1补仓触发：亏损 ≥ 0.5 × ATR
- L2补仓触发：亏损 ≥ 1.0 × ATR
- 补仓量：当前仓量 × 0.5
- 最大补仓次数：2次
- 补仓间隔：≥30分钟

### DOT黑名单
- 当前等级：{dot_blacklist_level}
- 黑名单等级：watch/warn/black
- black级：跳过所有新开仓
```

## 五、AI Agent 实现代码

```python
import anthropic
from anthropic import Anthropic

class TianyanAgent:
    """
    AI Agent「天眼」
    完整子代理，接收持仓数据，输出判断+执行命令
    """

    def __init__(self, api_key: str):
        self.client = Anthropic(api_key=api_key)
        self.model = "claude-sonnet-4-7"
        self.max_tokens = 2048

    SYSTEM_PROMPT = """你是加密货币持仓监控AI代理，代号「天眼」。
你拥有完整执行权，人类已授权你直接下发命令。
你的输出直接触发执行，不需要人类审批。"""

    def analyze(self, positions: list, dot_blacklist: dict, dca_status: dict) -> list:
        """
        分析所有持仓，返回执行决策列表
        """
        prompt = self._build_prompt(positions, dot_blacklist, dca_status)

        response = self.client.messages.create(
            model=self.model,
            max_tokens=self.max_tokens,
            system=self.SYSTEM_PROMPT,
            messages=[{"role": "user", "content": prompt}]
        )

        raw = response.content[0].text
        # 提取JSON（可能带有markdown代码块）
        import re, json
        match = re.search(r'\{[\s\S]*\}', raw)
        if match:
            return json.loads(match.group())
        return None

    def _build_prompt(self, positions, dot_blacklist, dca_status):
        pos_str = json.dumps(positions, ensure_ascii=False, indent=2)
        dot_str = json.dumps(dot_blacklist, ensure_ascii=False)
        dca_str = json.dumps(dca_status, ensure_ascii=False)
        return f"""输入数据：

positions = {pos_str}

dot_blacklist = {dot_str}

dca_status = {dca_str}

请严格按以下JSON格式输出每个持仓的决策（返回数组）：
[
  {{
    "trade_id": 数字,
    "port": 数字,
    "pair": "字符串",
    "analysis": {{
      "alert_type": "wrong_direction|premature_entry|normal",
      "wrong_signals": ["信号1", "信号2"],
      "sr_confidence": "high|medium|low",
      "sr_levels": {{"support": 数字, "resistance": 数字}},
      "atr_14": 数字
    }},
    "score": 数字(0-100),
    "confidence": 数字(0-100),
    "action": {{
      "type": "observe|half_close|full_close|dca_l1|dca_l2",
      "reason": "决策理由（中文，20字以内）"
    }},
    "execute": true或false,
    "feishu_alert": true或false,
    "feishu_msg": "飞书消息内容（中文，包含emoji）"
  }}
]

决策规则：
- 评分<40：1轮触发，execute=true
- 评分40-59：2轮触发，execute=true
- 评分≥60：3轮触发，execute=true
- normal类型：execute=false

补仓参数：
- ATR(14)：取输入数据中 atr_14
- L1触发：亏损 ≥ 0.5 × ATR
- L2触发：亏损 ≥ 1.0 × ATR（需先有L1）
- 最大补仓次数：2次

只输出JSON数组，不要输出其他内容。"""
```

```python
def _auto_early_exit_loop():
    """
    后台常驻线程，每5分钟触发一次完整流程：
    1. 获取所有持仓
    2. 获取 OHLCV + 计算 S/R + ATR
    3. 调用 AI Agent 分析
    4. 执行决策（自动，无需审批）
    5. 发送飞书通知
    6. 记录准确率
    """
    import anthropic

    agent = TianyanAgent(api_key=os.environ["ANTHROPIC_API_KEY"])

    while _em_state["running"]:
        try:
            # 1. 获取所有持仓
            positions = _fetch_all_positions()

            # 2. 获取 OHLCV + 计算 S/R + ATR
            enriched = []
            for pos in positions:
                ohlcv = _fetch_ohlcv(pos)
                atr = _calculate_atr(ohlcv, period=14)
                sr = _calculate_sr_levels(ohlcv)
                pos["ohlcv"] = ohlcv
                pos["atr_14"] = atr
                pos["sr_levels"] = sr
                enriched.append(pos)

            # 3. DOT黑名单状态
            dot_bl = _get_dot_blacklist()

            # 4. DCA状态
            dca_st = _get_dca_status()

            # 5. AI Agent 分析
            decisions = agent.analyze(enriched, dot_bl, dca_st)

            # 6. 执行决策（自动）
            for decision in decisions:
                if decision.get("execute"):
                    _execute_decision(decision)
                    _log_alert(decision)

                # 7. 发送飞书
                if decision.get("feishu_alert"):
                    _send_feishu(decision)

            # 8. 更新准确率统计
            _update_accuracy()

        except Exception as e:
            print(f"[天眼] 扫描异常: {e}")

        time.sleep(SCAN_INTERVAL)  # 5分钟


def _execute_decision(decision: dict):
    """执行决策，调用机器人API"""
    action_type = decision.get("action", {}).get("type")
    if action_type == "observe":
        return  # 无需执行

    trade_id = decision["trade_id"]
    port = decision["port"]

    auth = BOT_AUTH.get(port)
    base_url = f"http://localhost:{port}"

    if action_type == "half_close":
        # 半仓
        resp = requests.post(
            f"{base_url}/api/monitor/close",
            json={"tradeid": trade_id, "amount": "50%"},

        )

    elif action_type == "full_close":
        # 全平
        resp = requests.post(
            f"{base_url}/api/monitor/close",
            json={"tradeid": trade_id},
            auth=auth, timeout=10
        )

    elif action_type in ("dca_l1", "dca_l2"):
        # 补仓
        resp = requests.post(
            f"{base_url}/api/monitor/add_position",
            json={"tradeid": trade_id, "amount": "50%"},

    if resp and resp.ok:
        result = resp.json()
        decision["exec_success"] = True
        decision["exec_result"] = result
        print(f"[天眼] 执行成功 {port} {action_type} trade_id={trade_id}")
    else:
        decision["exec_success"] = False
        print(f"[天眼] 执行失败 {port} {action_type} trade_id={trade_id}")
```

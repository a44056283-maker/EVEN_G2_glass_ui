## 一、决策卡片系统概述

### 1.1 卡片评级体系

| 评级 | 分值区间 | 边框颜色 | 风险等级 | 触发轮次 |
|------|---------|---------|---------|---------|
| 🔴 临界（critical） | score < 35 | 红色 | 极高危 | 0轮（立即执行） |
| 🟠 高危（high） | 35 ≤ score < 40 | 橙色 | 高危 | 1轮 |
| 🟡 中危（medium） | 40 ≤ score < 60 | 黄色 | 中危 | 2轮 |
| 🟢 低危（low） | score ≥ 60 | 绿色 | 低危 | 3轮 |

### 1.2 预警类型

| 类型 | 标签 | 说明 |
|------|------|------|
| wrong_direction | 方向错误 | 趋势判断与持仓方向相悖 |
| premature_entry | 过早入场 | 入场时机不成熟 |
| normal | 正常 | 无明显异常 |

## 二、轮次门控规则

### 2.1 轮次触发机制

```
评分 < 35    → 立即执行（0轮，不等待）
35 ≤ score < 40  → 1轮触发后执行
40 ≤ score < 60  → 2轮触发后执行
score ≥ 60   → 3轮触发后执行
```

### 2.2 方向错误信号（满足≥3条触发，每条20分）

1. **RSI极端区+浮亏>1%**
   - 做多：RSI > 70
   - 做空：RSI < 30

2. **RSI方向背离**
   - 做多：RSI < 50
   - 做空：RSI > 50

3. **S/R突破**
   - 跌破支撑 > 1%
   - 突破阻力 > 1%

4. **量能萎缩**
   - 连续3根K线量比 < 1.0

5. **铁律违反**
   - 做多：不在支撑位入场
   - 做空：不在压力位入场

### 2.3 过早入场信号（满足≥2条触发，每条25分）

1. **入场距S/R > 3%**（中间地带入场）
2. **入场量比 < 2.5x**
3. **R3引擎判定过早**
4. **入场RSI非极端值**

## 四、动作类型话术

### 4.1 保持观察（observe）

**话术**：👀 保持观察

**卡片显示**：

**适用场景**：
- 评分 ≥ 60
- 无明显方向错误
- 过早入场但可观察

### 4.2 半仓平（half_close）

**话术**：✂️ 半仓平

**卡片显示**：


**适用场景**：
- 评分 40-79
- 方向错误信号出现
- 需要减仓控制风险

---

### 4.3 全仓平（full_close）

**话术**：🛑 全仓平

**卡片显示**：
```
决策: 🛑 全仓平
原因: [AI决策理由]

轮次进度: ██████████ 立即 🔴
```

**适用场景**：
- 评分 < 40
- 多重铁律违反
- 趋势完全破位

---

### 4.4 L1补仓（dca_l1）

**话术**：📈 L1补仓

**卡片显示**：
```
决策: 📈 L1补仓
原因: [AI决策理由]

轮次进度: ████░░░░░░ 2/3 ✅
```

**补仓参数**：
- 触发条件：亏损 ≥ 0.5 × ATR(14)
- 补仓量：当前仓量 × 50%
- 最大次数：2次

### 4.5 L2补仓（dca_l2）

**话术**：📈📈 L2补仓

**卡片显示**：
```
决策: 📈📈 L2补仓
```

## 十、干预动作策略详解

### 10.1 方向错误干预流程

```
任何评分 方向错误
    │
    ├── 先执行 half_close（半仓平）
    │
    └── 等待风向确认
            │
            ├── 续亏确认 + 反弹 > 1%
            │       └── 执行 full_close（全平）
            │
            └── 反弹确认 + 反弹 > 0.5%
                    └── 执行 dca_l1（补仓）
```

### 10.2 过早入场干预流程

```
评分 60-79（过早入场）
    │
    └── observe（保持观察）
            │
            ├── 观察期间恶化
            │       └── 评分降至40-59
            │               └── 监控补仓线
            │
            └── 评分进一步下降 < 40
                    └── half_close + 补仓线设置
```

### 10.3 高危干预（评分 < 40 + 多重违反）

```
评分 < 40 + 多重铁律违反
    │
    └── full_close（立即全平）
            │
            └── 标记高危，记录误报
```

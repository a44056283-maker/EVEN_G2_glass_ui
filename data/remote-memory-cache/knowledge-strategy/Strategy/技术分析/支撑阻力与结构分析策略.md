## 一，核心概念

### 1.1 支撑阻力的本质

**支撑**：价格下跌时遇到"地板"，需求涌入阻止下跌
**阻力**：价格上涨时遇到"天花板"，供应涌入阻止上涨
**本质**：供需拐点——价格在某一价位形成暂时平衡

### 1.2 支撑阻力的三大类型

| 类型 | 重要程度 | 形成原因 |
|------|---------|---------|
| 价格密集区 | ⭐⭐⭐ 最强 | 多空充分博弈 |
| 前期高低点 | ⭐⭐⭐ 最强 | 记忆效应 |
| 均线/整数关口 | ⭐⭐ 一般 | 算法/心理 |

## 二，K线形态核心反转信号

### 2.1 锤子线（Hammer）— 底部反转

**形态**：
```
       ┌──┐
       │  │ ← 上影线短
       │  │
       │  │ ← 下影线≥实体2倍
  ─────┴──┴─────
```

**判断标准**：
- 出现在下跌趋势末端（至少5根K线以上）
- 下影线长度 ≥ 实体2倍
- 上影线尽量短（< 实体10%）

**信号强度**：
- 锤子线 + 放量 = 更强
- 锤子线 + 跳空高开 = 最强

---

### 2.2 吞没形态（Engulfing）— 强反转

**看涨吞没**：
```
┌────────┐
│  阴线  │ ← 阴线（跌）
└────────┘
┌────────────┐
│   阳线    │ ← 阳线，完全吞没前一根
└────────────┘
```

**判断标准**：
- 出现在明显趋势末端
- 第二根K线实体必须完全吞没第一根

**看跌吞没**：反向操作

---

### 2.3 十字星（Doji）— 犹豫信号

**形态**：开盘≈收盘，几乎无实体

**类型**：
- 普通十字星：多空犹豫
- 长脚十字星：极度犹豫
- 蜻蜓十字星：下影线长 → 可能反转
- 墓碑十字星：上影线长 → 可能反转

**关键**：结合位置判断，孤立无意义

---

### 2.4 启明之星 + 黄昏之星

**启明之星（底部反转）**：
```
第1根：大阴线（跌） ← 下跌确认
第2根：小K线（十字/锤子）← 多空僵持
第3根：大阳线（涨）← 多头反攻
```
信号强度：⭐⭐⭐⭐⭐

**黄昏之星（顶部反转）**：反向

## 三，结构分析流程

```
Step 1: 找关键支撑阻力位
         ↓
   水平线（前期高低点）
   趋势线（斜率判断）
   均线（MA20/MA60）
         ↓
Step 2: 判断当前价格位置
         ↓
   在支撑附近？→ 观察是否出现看涨K线形态
   在阻力附近？→ 观察是否出现看跌K线形态
   在中间？→ 观望
         ↓
Step 3: 结合量能确认
         ↓
   放量突破 = 真突破 → 顺势交易
   缩量突破 = 假突破 → 可能反转
         ↓
Step 4: 制定交易计划
         ↓
   支撑位 + 看涨形态 + 放量 = 买入
   阻力位 + 看跌形态 + 放量 = 卖出
```

## 四，威科夫 + 支撑阻力 综合应用

| 威科夫概念 | 支撑阻力应用 | 信号 |
|-----------|------------|------|
| Spring | 跌破支撑后快速拉回 | 支撑有效，多头信号 |
| UT | 突破阻力后快速回落 | 阻力有效，空头信号 |
| SOS | 在支撑位放量反弹 | 做多 |
| SOW | 在阻力位放量回落 | 做空 |
| BU | 突破后回踩原阻力 | 回踩确认，做多 |
| LPS | 回调到前支撑附近 | 做多机会 |

## 六，实战策略

### 策略1：支撑位做多 / 阻力位做空

```
条件：
  ① 价格触及支撑位
  ② 出现看涨K线形态（锤子/吞没/启明之星）
  ③ 成交量放大

买入计划：
  入场：支撑位附近收盘确认后
  止损：支撑位下方1-2%
  止盈：前高阻力位或2:1盈亏比

风险管理：
  止损幅度 ≤ 总资金2%
```

### 策略2：突破回踩交易

```
条件：
  ① 价格向上突破阻力位
  ② 回踩到原阻力位（现变支撑）
  ③ 缩量回踩 = 假跌 = 买入机会

对应：威科夫BU / 缠论三买
```

### 策略3：区间交易

```
适用：价格在水平区间内震荡

操作：
  区间下沿 + 看涨形态 = 买入
  区间上沿 + 看跌形态 = 卖出
  止损：区间外侧1-2%

对应：威科夫吸筹/派发区间 / 缠论中枢震荡
```

## 七，三维共振（最高胜率）

| 共振要素 | 说明 |
|---------|------|
| 支撑阻力位 | 价格在关键位反应 |
| K线形态 | 出现反转形态确认 |
| 威科夫/缠论 | 结构支持方向 |

**三维共振 = 高概率信号**

```
支撑位 + 锤子线 + 威科夫Spring = 最强买入信号
阻力位 + 吞没形态 + 威科夫UT = 最强卖出信号
```

## 八，误区提醒

| 误区 | 正确认知 |
|------|---------|
| 支撑位不会破 | ❌ 所有支撑都可能破，关键是破后是否快速拉回（Spring） |
| 跌破就做空 | ❌ 看整体形态，支撑下方可能有陷阱 |
| 越多越好 | ❌ 重叠的多个SR位 = 更强 |
| 精确到分 | ❌ 支撑阻力是"区域"，±0.5%合理 |

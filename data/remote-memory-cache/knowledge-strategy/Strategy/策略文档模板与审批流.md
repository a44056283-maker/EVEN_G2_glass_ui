## 二，策略文档核心字段

### 必填字段

| 字段 | 说明 |
|------|------|
| 策略名称 | 唯一标识 |
| 版本 | v主.次.修订 |
| 入场条件 | 技术指标+价格+量能 |
| 出场条件 | 止损+止盈+分批 |
| 仓位管理 | 单笔风险+杠杆+加仓规则 |
| 历史绩效 | 回测数据（至少3个月） |

### 选填字段

| 字段 | 说明 |
|------|------|
| 适用市场 | 趋势/震荡/突破 |
| 适用品种 | BTC/ETH/山寨币 |
| 注意事项 | 特殊市场条件 |

## 三，策略审批流程

```
策略制定 → 回测验证 → 模拟盘测试 → 实盘审批 → 执行监控
     ↓           ↓            ↓           ↓          ↓
   中书省      刑部        尚书省      尚书省     工部
```

### 各阶段要求

| 阶段 | 负责 | 要求 |
|------|------|------|
| 回测验证 | 中书省 | 3个月以上回测，胜率>40%，盈亏比>1.5 |
| 模拟盘 | 尚书省 | 1个月模拟，胜率接近回测80%以上 |
| 实盘审批 | 尚书省 | 审核全部文档，批准后方可执行 |
| 执行监控 | 工部 | 实时监控，发现异常立即上报 |

## 四，策略有效性检验

### 新策略上线前必须通过

1. **回测检验**
   - 周期：3个月+
   - 样本：多个市场环境（牛/熊/震荡）
   - 指标：胜率/盈亏比/最大回撤/夏普比率

2. **模拟盘检验**
   - 周期：1个月
   - 资金：真实账户10%资金
   - 目标：胜率接近回测80%

3. **实盘检验**
   - 仓位：从小到大渐进
   - 监控：每日复盘

## 五，策略淘汰机制

| 条件 | 动作 |
|------|------|
| 连续3个月胜率<30% | 暂停，复盘优化 |
| 连续5笔亏损 | 强制复盘 |
| 市场环境重大变化 | 重新评估适用性 |

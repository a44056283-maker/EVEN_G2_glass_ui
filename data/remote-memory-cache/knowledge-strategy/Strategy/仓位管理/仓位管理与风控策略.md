## 一、凯利公式 — 最优仓位计算

### 1.1 公式

```
f* = (p × b - q) / b

其中：
f* = 最优仓位比例
p = 胜率（盈利概率）
q = 1 - p（亏损概率）
b = 盈亏比（平均盈利/平均亏损）
```

### 1.2 Python实现

```python
def kelly_criterion(win_rate: float, avg_win: float, avg_loss: float) -> dict:
    """
    凯利公式计算最优仓位

    Args:
        win_rate: 胜率 (0-1)
        avg_win: 平均盈利金额
        avg_loss: 平均亏损金额

    Returns:
        最优仓位比例和建议
    """
    p = win_rate
    q = 1 - p
    b = avg_win / avg_loss if avg_loss > 0 else 0

    if b <= 0:
        return {"f_star": 0, "advice": "盈亏比为0，无法计算"}

    kelly = (p * b - q) / b

    # 实际建议用半凯利（降低波动）
    half_kelly = kelly / 2
    quarter_kelly = kelly / 4

    return {
        "p": p,
        "q": q,
        "b": b,
        "kelly_full": max(0, kelly),
        "kelly_half": max(0, half_kelly),
        "kelly_quarter": max(0, quarter_kelly),
        "advice": f"建议仓位: {half_kelly*100:.1f}% (半凯利)"
    }

# 示例
result = kelly_criterion(win_rate=0.4, avg_win=1000, avg_loss=500)
print(f"最优仓位: {result['kelly_half']*100:.1f}%")
```

### 1.3 实战应用

| 场景 | 胜率p | 盈亏比b | 全凯利 | 半凯利 | 建议仓位 |
|------|-------|--------|--------|--------|---------|
| 趋势跟踪 | 0.38 | 3.0 | 24% | 12% | 10% |
| 均值回归 | 0.55 | 1.5 | 37% | 18% | 15% |
| 突破策略 | 0.42 | 2.2 | 31% | 15% | 12% |

**注意**：加密货币高波动市场，建议用1/4凯利或更低

## 二、ATR动态止损

### 2.1 ATR定义

ATR（Average True Range）= N日内真实波幅的平均值

**True Range (TR)**：
```
TR = max(当日高点-当日低点, |当日高点-前收盘|, |当日低点-前收盘|)
```

### 2.2 ATR止损公式

```
止损价位 = 入场价 - N × ATR

其中N的经验值：
- 短期止损（短线）：1.5-2×ATR
- 中期止损（中线）：2-3×ATR
- 长期止损（长线）：3-5×ATR
```

### 2.3 Python实现

```python
def calculate_atr(klines, period=14):
    """计算ATR"""
    tr_list = []
    for i in range(1, len(klines)):
        high = klines[i]['high']
        low = klines[i]['low']
        prev_close = klines[i-1]['close']

        tr = max(
            high - low,
            abs(high - prev_close),
            abs(low - prev_close)
        )
        tr_list.append(tr)

    if len(tr_list) < period:
        return None

    atr = sum(tr_list[-period:]) / period
    return atr

def atr_stop_loss(entry_price, atr, atr_mult=2.0, direction='long'):
    """ATR动态止损"""
    if direction == 'long':
        stop = entry_price - atr_mult * atr
    else:
        stop = entry_price + atr_mult * atr
    return stop
```

## 三、强平价计算

### 3.1 强平价公式

```
多仓强平价 = 开仓价 × (1 - 1/杠杆倍数)
空仓强平价 = 开仓价 × (1 + 1/杠杆倍数)
```

### 3.2 不同杠杆的强平距

| 杠杆 | 维持保证金率 | 多仓强平距离开仓价 |
|------|------------|-----------------|
| 5x | 20% | -20% |
| 10x | 10% | -10% |
| 20x | 5% | -5% |
| 50x | 2% | -2% |
| 100x | 1% | -1% |

### 3.3 Python实现

```python
def calculate_liquidation_price(entry_price, leverage, direction='long'):
    """计算强平价"""
    maintenance_rate = 1 / leverage

    if direction == 'long':
        liq_price = entry_price * (1 - maintenance_rate)
    else:
        liq_price = entry_price * (1 + maintenance_rate)

    distance_pct = maintenance_rate * 100
    return {
        'entry_price': entry_price,
        'leverage': leverage,
        'direction': direction,
        'liquidation_price': round(liq_price, 2),
        'distance_from_entry_pct': round(distance_pct, 2),
        'safe_entry_buffer': round(distance_pct * 0.5, 2)  # 安全缓冲
    }

# 示例
result = calculate_liquidation_price(67000, 10, 'long')
# 强平价 = 67000 × 0.9 = $60,300
# 距离 = 10%
```

## 四、VaR（风险价值）

### 4.1 VaR定义

在给定置信水平下，在特定时间段内可能遭受的最大损失

```
VaR(95%, 1日) = 账户 × 波动率 × 1.65
```

### 4.2 Python实现

```python
def calculate_var(account_value, volatility_daily, confidence=0.95):
    """计算VaR"""
    # Z分数
    if confidence == 0.95:
        z_score = 1.65
    elif confidence == 0.99:
        z_score = 2.33
    else:
        z_score = 1.65

    var = account_value * volatility_daily * z_score
    return {
        'var_pct': round(volatility_daily * z_score * 100, 2),
        'var_dollar': round(var, 2),
        'confidence': f"{confidence*100}%"
    }
```

## 五、仓位配置策略

### 5.1 固定比例法

```
单笔仓位 = 账户 × 固定比例

示例：
账户 = $10,000
固定比例 = 10%
单笔仓位 = $1,000
```

### 5.2 波动率调整法

```
调整后仓位 = 固定仓位 × (目标波动率 / 标的波动率)

标的ATR% = ATR / 价格 × 100
目标波动率 = 1%
仓位 = $1,000 × (1% / 标的ATR%)
```

### 5.3 风险预算分配

```
账户 = $10,000
单笔最大风险 = 2%
单笔最大亏损 = $200

止损点数 × 每点价值 = $200
```

| 规则 | 阈值 | 动作 |
|------|------|------|

## 八、最大回撤控制（新增 2026-04-05）

### 8.1 最大回撤定义

```
Max Drawdown = (Peak - Trough) / Peak × 100%
```

### 8.2 回撤与回本对照

| 最大回撤 | 回本所需盈利 |
|---------|------------|
| 10% | 11.1% |
| 20% | 25.0% |
| 30% | 42.9% |
| 50% | 100.0% |
| 75% | 300.0% |

**核心：回撤越大，回本难度指数级增长**

### 8.3 风险预算层级

```
总风险预算（年度）
├── 月度风险预算（≤15%）
│   └── 周度风险预算（≤8%）
│       └── 日度风险预算（≤5%）
│           └── 单笔风险预算（≤2%）
```

### 8.4 动态仓位调整

```
实际仓位 = f*(凯利) × 风险调整系数

风险调整系数：
  - 回撤<5%：1.0（正常）
  - 回撤5-10%：0.5（降50%）
  - 回撤10-15%：0.25（降75%）
  - 回撤>15%：0（暂停交易）
```

### 8.5 Python实现

```python
class DrawdownMonitor:
    def __init__(self, max_drawdown=0.20, daily_limit=0.05, weekly_limit=0.08):
        self.peak = 0
        self.max_drawdown_limit = max_drawdown
        self.daily_loss_limit = daily_limit
        self.weekly_loss_limit = weekly_limit

    def update(self, current_equity: float) -> dict:
        if current_equity > self.peak:
            self.peak = current_equity
        dd = (self.peak - current_equity) / self.peak
        status = "CRITICAL" if dd >= self.max_drawdown_limit else "NORMAL"
        return {"drawdown_pct": dd*100, "status": status, "can_trade": status=="NORMAL"}

    def get_position_reduction(self, equity, base_pos):
        dd = (self.peak - equity) / self.peak if self.peak else 0
        if dd < 0.05: return base_pos
        elif dd < 0.10: return base_pos * 0.5
        elif dd < 0.15: return base_pos * 0.25
        else: return 0
```

## 三、仓位管理核心公式

> ⚠️ 详细内容见 `Knowledge/仓位管理/凯利公式与ATR动态止损实战指南.md`（2026-04-04 更新）

### 3.1 凯利公式

```
f* = (p × b - q) / b

其中：
f* = 最优仓位比例
p = 胜率（盈利概率）
q = 1 - p（亏损概率）
b = 盈亏比（平均盈利/平均亏损）

示例：
p = 0.45（45%胜率）
b = 2.0（盈亏比2:1）
f* = (0.45 × 2 - 0.55) / 2 = 0.175 = 17.5%

币圈实盘建议：
- 全凯利 → 波动太大，用半凯利（×50%）
- 高波动币圈 → 建议用1/4凯利或更低
- 参考：户部当前胜率61.3%，半凯利约28%，实盘建议≤15%
```

### 3.2 ATR动态止损

```
做多止损 = 入场价 - (2 × ATR)
做空止损 = 入场价 + (2 × ATR)

做多止盈 = 入场价 + (4 × ATR)（2:1盈亏比）
做空止盈 = 入场价 - (4 × ATR)
```

### 3.2 风险预算

```
单笔最大损失 = 账户 × 风险比例

账户 = $10,000
风险比例 = 2%
单笔最大损失 = $200

止损点数 = $200 / 每点价值

示例：
止损 = 100点
每点价值 = $10
仓位 = $200 / ($100 × 1手) = 0.02手
```

### 3.3 ATR动态仓位

```
仓位 = 账户 × 风险比例 / (ATR × 每点价值)

账户 = $10,000
风险比例 = 2%
ATR = 150点
每点价值 = $10

仓位 = $10,000 × 0.02 / (150 × $10) = 0.13手
```

## 四、趋势跟踪策略

### 4.1 均线交叉策略

**原理**：短期均线上穿长期均线 = 买入信号

**参数**：
- 短期均线：20日EMA
- 长期均线：60日EMA

**信号**：
```
买入：EMA20 上穿 EMA60，且价格在EMA20之上
卖出：EMA20 下穿 EMA60，或价格在EMA20之下
```

**回测表现**：
- 标的：BTC 2020-2024
- 胜率：38%
- 盈亏比：3.2x
- 年化收益：45%

### 4.2 趋势线突破策略

**原理**：价格突破趋势线 = 趋势反转/加速

**参数**：
- 趋势线：连接2个显著高点/低点
- 确认：收盘价站稳趋势线外

**信号**：
```
买入：价格上破下降趋势线，且成交量放大
卖出：价格下破上升趋势线，且成交量放大
```

### 4.3 均线+ATR止损策略

**原理**：趋势跟踪 + 动态止损

**参数**：
- 入场：EMA50 金叉 EMA200
- 止损：入场价 - 2×ATR
- 止盈：止损的3倍（盈亏比3:1）

**代码示例**：

```python
class TrendFollower:
    def __init__(self, fast_ema=20, slow_ema=60, atr_period=14):
        self.fast_ema = fast_ema
        self.slow_ema = slow_ema
        self.atr_period = atr_period
    
    def on_bar(self, df):
        # 计算指标
        df['ema_fast'] = df['close'].ewm(span=self.fast_ema).mean()
        df['ema_slow'] = df['close'].ewm(span=self.slow_ema).mean()
        df['atr'] = self._calc_atr(df)
        
        # 均线交叉信号
        if df['ema_fast'].iloc[-1] > df['ema_slow'].iloc[-1] and \
           df['ema_fast'].iloc[-2] <= df['ema_slow'].iloc[-2]:
            return 'BUY'
        
        if df['ema_fast'].iloc[-1] < df['ema_slow'].iloc[-1] and \
           df['ema_fast'].iloc[-2] >= df['ema_slow'].iloc[-2]:
            return 'SELL'
        
        return 'HOLD'
    
    def get_stop_loss(self, entry_price, atr, direction):
        if direction == 'BUY':
            return entry_price - 2 * atr
        else:
            return entry_price + 2 * atr
```

## 五、均值回归策略

### 5.1 布林带策略

**原理**：价格触及布林带极端位置后回归均值

**参数**：
- 中轨：20日均线
- 上轨：中轨 + 2×标准差
- 下轨：中轨 - 2×标准差

**信号**：
```
买入：价格触及下轨，且RSI < 30
卖出：价格触及上轨，或RSI > 70
```

### 5.2 RSI均值回归策略

**原理**：RSI超卖后反弹

**参数**：
- RSI周期：14
- 超卖线：30
- 超买线：70

**信号**：
```
买入：RSI < 30，且价格触及支撑位
卖出：RSI > 70，或RSI死叉50
```

### 5.3 斐波那契回撤策略

**原理**：价格在斐波那契回撤位企稳

**参数**：
- 关键回撤位：38.2%, 50%, 61.8%

**信号**：
```
买入：价格在61.8%回撤位企稳，出现反转K线
止损：跌破61.8%回撤位
目标：前高
```

## 七、网格交易策略

### 7.1 经典等差网格

**原理**：在固定价格区间等间距下单

**参数**：
- 网格数量：10格
- 网格间距：价格 × 1%
- 每格仓位：固定

**代码示例**：

```python
class GridTrader:
    def __init__(self, grid_count=10, grid_spacing_pct=1.0, position_per_grid=0.01):
        self.grid_count = grid_count
        self.grid_spacing_pct = grid_spacing_pct / 100
        self.position_per_grid = position_per_grid  # BTC
    
    def create_grid(self, center_price):
        """创建网格"""
        self.grids = []
        half_count = self.grid_count // 2
        
        for i in range(-half_count, half_count + 1):
            price = center_price * (1 + i * self.grid_spacing_pct)
            self.grids.append({
                'price': price,
                'buy_filled': False,
                'sell_filled': False,
                'position': 0
            })
    
    def on_price_update(self, current_price):
        """价格更新时触发"""
        for grid in self.grids:
            if not grid['buy_filled'] and current_price <= grid['price']:
                # 触发买入
                grid['buy_filled'] = True
                grid['position'] = self.position_per_grid
                yield ('BUY', grid['price'], self.position_per_grid)
            
            if grid['buy_filled'] and not grid['sell_filled'] and \
               current_price >= grid['price'] * (1 + self.grid_spacing_pct):
                # 触发卖出
                grid['sell_filled'] = True
                grid['position'] = 0
                yield ('SELL', grid['price'] * (1 + self.grid_spacing_pct), self.position_per_grid)
```

## 八、套利策略

### 8.1 跨所搬砖

**原理**：A所价格 < B所价格，搬砖套利

**计算**：
```
利润 = (B所价格 - A所价格) × 数量 - 手续费

示例：
币安BTC = $66,000
OKX BTC = $66,050
数量 = 0.1 BTC
手续费 = 0.1%
利润 = ($66,050 - $66,000) × 0.1 - $66,050 × 0.1% × 2
    = $5 - $0.13 = $4.87
```

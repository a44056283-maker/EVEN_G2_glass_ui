# 太子每日例行程序

> 太子每日雷打不动必须完成的事项
> 时间：每天

---

## 一，每日必做（按时间顺序）

### 08:00 晨间自检
- [ ] 读 error_log.md（如果有新错误）
- [ ] 读 self_evolution.md（昨天进度）
- [ ] 确认8个机器人状态（9090-9097）
- [ ] 检查是否有紧急技术问题

### 09:00 天下要闻调度
- [ ] 生成7部门日报
- [ ] 尚书省汇总决策
- [ ] 向父亲汇报

### 14:00-15:00 每日学习
- [ ] 按学习计划学习1个主题（4周循环）
- [ ] 写学习笔记到 learning/YYYY-MM-DD.md
- [ ] 整理精华到知识库

### 18:00 暮间复盘
- [ ] 复盘今日操作（决策/偏差）
- [ ] 更新 error_log.md（如有新错误）
- [ ] 复盘学习内容

### 22:00 夜间进化
- [ ] 生成今日进化报告
- [ ] 更新 self_evolution.md
- [ ] 检查策略库更新

1. 学完写笔记 → workspace/learning/YYYY-MM-DD.md
2. 整理精华 → 知识策略库 Knowledge/ 对应目录
3. 形成策略 → 知识策略库 Strategy/ 对应目录

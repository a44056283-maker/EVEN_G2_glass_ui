## 二、新增文件清单

### 前端文件（static/tabs/）

| 文件路径 | 用途 |
|---------|------|
| `static/tabs/ai_agents.html` | AI子代理总览面板（主页） |
| `static/tabs/ai_market_radar.html` | 市场雷达分析Tab |
| `static/tabs/ai_alert_analytics.html` | 预警准确率分析Tab |
| `static/tabs/ai_dot_blacklist.html` | DOT黑名单管理Tab |
| `static/tabs/ai_position_optimizer.html` | 仓位优化Tab |

### 后端文件（freqtrade_console/）

| 文件路径 | 用途 |
|---------|------|
| `freqtrade_console/ai_agents/` | AI子代理模块目录 |
| `freqtrade_console/ai_agents/__init__.py` | 模块入口 |
| `freqtrade_console/ai_agents/market_radar.py` | P0-1 市场雷达子代理 |
| `freqtrade_console/ai_agents/alert_analytics.py` | P0-3 预警分析子代理 |
| `freqtrade_console/ai_agents/dot_blacklist.py` | P1-2 DOT黑名单子代理 |
| `freqtrade_console/ai_agents/position_optimizer.py` | P1-1 仓位优化子代理 |
| `freqtrade_console/ai_agents/ai_config.py` | AI子代理配置管理 |

### API 扩展（console_server.py 新增）

| 端点 | 方法 | 用途 |
|------|------|------|
| `/api/ai/status` | GET | AI子代理总状态 |
| `/api/ai/market_radar` | GET | 市场雷达分析结果 |
| `/api/ai/market_radar/refresh` | POST | 强制刷新市场雷达 |
| `/api/ai/alert_analytics` | GET | 预警准确率报表 |
| `/api/ai/alert_analytics/report` | GET | 详细准确率报告 |
| `/api/ai/dot_blacklist` | GET | 黑名单状态 |
| `/api/ai/dot_blacklist` | POST | 手动调整黑名单 |
| `/api/ai/position_config` | GET/POST | 仓位优化配置 |
| `/api/ai/agents/toggle` | POST | 开关某个子代理 |
